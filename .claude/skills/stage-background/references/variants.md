# 環境バリアント・カタログ（ロール表／アートスタイル／プリセット）

ステージ背景の「**同じ場所が毎回違う姿になる**」を体系化した軸。SKILL の STEP2 でここから振る（ランダム or ユーザー指定）。
出典の視覚言語＝著者の参考サンプル（`C:\Users\ryuya\OneDrive\ドキュメント\アニメ背景` の `future_100_year_style_samples`／`fuji_sakura_100_year_samples`／`battle_background_samples`）。
正典の根拠＝[dungeon_design.md §1.5-3](../../../../docs/settings/dungeon_design.md)（固有現象＝同一性を保ち、天候・時間・歪み方はランダム）。

## 0. ロールの仕方

- **同一性（固定）**＝実在地ベース＋固有現象は**振らない**。下記5軸＋スタイルだけを振る。
- ランダムに振る例（Bash）：
  ```bash
  shuf -e 昼 夕 夜 薄明 -n1        # 時間帯
  shuf -e 晴 曇 雨 霧 雪 嵐 -n1    # 天候
  shuf -e なし 森化 水没 氷結 砂漠化 崩落反転 -n1  # 地形変容
  ```
- ユーザーが「夜」「荒廃」「森と化す」「水中」等を指定したらそれを採用（ロールより優先）。
- 同じ場所で**複数バリアントを並べて出す**と「毎回違う姿」を可視化できる（バリアントシート）。

## 1. 時間帯（time of day）

| 値 | 画の核 |
|---|---|
| 昼 | 順光・空が広い・影が短い。基準 |
| 夕（マジックアワー） | 暖色グラデ・長い影・逆光。劇場的（参考＝02/10） |
| 夜 | 暗部主体・人工光が主役・反射 |
| 薄明（夜明け前/黄昏） | 青〜紫の静けさ・霧と相性 |

## 2. 天候（weather）

| 値 | 画の核 |
|---|---|
| 晴 | 抜けの良い空・コントラスト高 |
| 曇 | 拡散光・彩度低め・静か（参考＝01） |
| 雨 | 路面反射・濡れ・**「地下なのに雨」=ありえない一点の定番** |
| 霧 | 奥行きを消す・気配・不穏 |
| 雪 | 減色・静寂・足跡 |
| 嵐 | 暴風・荒れた海・荒廃と相性（参考＝06） |

## 3. トーン／エネルギー（mood）

| 値 | 画の核 | 参考 |
|---|---|---|
| 静謐 | 生活の温度・穏やか・スローライフ情緒 | 01/08 |
| エネルギーに満ちる | 発光・粒子・レンズフレア・高彩度・活気 | 04/05 |
| 荒廃 | 廃墟化・苔/蔦・崩れ・湿った青暗さ | 06 |
| 幻想 | 淡い光の靄・非現実の色・夢 | 08/10 |

## 4. 地形変容（terrain mutation）★深層ならではの大変化

| 値 | 画の核 |
|---|---|
| なし | 実在地の歪み程度（基準） |
| **森化** | 建物が緑に呑まれる・蔦と巨木・自然に還る（参考＝fuji_sakura 06 overgrown） |
| **水没／水中ステージ** | 街が水に沈む・水中光・気泡・揺らぎ・浮遊（参考＝fuji_sakura 02 river 系の発展） |
| 氷結 | 凍てつく・氷の反射・吐く息 |
| 砂漠化 | 乾いた砂・風紋・埋もれる構造物 |
| 崩落・反転 | 重力の歪み・宙に浮く破片・天地反転（深層の演算崩れ） |

## 5. ステージ特性＝固有現象（stage trait）★master の記憶のにじみ

> その場所の「癖」。**同一性の一部**だが、強弱・出方は振れる。任意で1つ重ねる。

| 値 | 画の核 |
|---|---|
| サイバーパンク化 | ネオン・電飾・反射する路面（参考＝07）。"ステージ特性が追加"の例 |
| 記録の雨 | 味のない雨・地面が濡れない（試作の三層） |
| 止まった人波 | 群衆が静止・時間が淀む |
| ループ廊下／戻る道 | 同じ景が繰り返す・出口がずれる |
| 逆さ重力／浮遊 | 物が落ちない・上下が曖昧 |
| 巨大化／縮小 | スケールが狂う（天井へ伸びる階段 等） |

## 7. ユーザー・プリセット（著者の指定例 → 値）

| 指定（著者） | 採用する値 |
|---|---|
| 日中 | 時間帯=昼／天候=曇 or 晴／スタイル=01 |
| 夜 | 時間帯=夜／スタイル=02 or 07 |
| エネルギーにあふれている | トーン=エネルギーに満ちる／スタイル=04 |
| 荒廃 | トーン=荒廃／天候=嵐／スタイル=06 |
| ステージ特性が追加（サイバーパンク） | ステージ特性=サイバーパンク化／時間帯=夜／スタイル=07 |
| 森と化す | 地形変容=森化 |
| 水中ステージ | 地形変容=水没／水中ステージ |

## 8. プロンプト雛形（英語・1枚分）

```text
Anime background art, [ART STYLE]. Establishing wide shot (cinematic, no characters).
Scene identity (KEEP CONSISTENT): [real-place base — e.g. a warped deep-layer version of a coastal Yokohama bridge leading to a hillside town], with its signature phenomenon: [固有現象].
Environment variant (this render): time = [time], weather = [weather], mood = [tone], terrain mutation = [terrain], stage trait = [trait].
One impossible detail that says "this is not the real world" WITHOUT any text: [ありえない一点].
Strict rules: absolutely NO readable text, NO letters, NO numbers, NO logos, NO UI/HUD, no real brand names. No people unless asked. Light modern-anime tone unless the style says otherwise.
```

- 縦横：establishing＝16:9（シネマ）／seedance 連携＝縦9:16。
- バリアントシート＝「1枚に複数バリアントをタイル状、隅に番号のみ」。
