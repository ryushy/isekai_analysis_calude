## 1. ログライン

2050年・近未来。世界は人知れず巨大な**分散バージョン管理システム**であり、人の一生はリポジトリ、技能は自ら書いた関数だった。新世界の native 世代として生まれた主人公は、自身の**消せない"異常な commit"**ゆえに、偽りの世界の中心(origin)を巡る覇権戦争へ巻き込まれていく。

## 2. 一行コンセプト

> **「現実そのものが Git だったら」** ── 戦えば世界が書き換わり、勝てば歴史を deploy する、コードと加速の異世界バトル。

## 3. 何が新しいか（USP）

1. **仮想と現実が分離していない**：仮想層＝現実のソースコード。**書けば現実が変わる(deploy)**。加速バトルもの（アクセル・ワールド）を"現実改変"まで踏み込ませた版。
2. **力＝数値ではなく literacy**：現実のコードをどれだけ読み書きできるか。import専門＜合成＜自作・改変。**インフレしない強さ**。
3. **技＝自作の関数・クラス・コンボ・ダンジョン徒弟**。修行の成果が自 repo に commit され技能化。名前は自由＝キャラの"コードの書き味"が個性になる。
4. **トランスメディア前提**：技名コール・HUD・色分け陣営・一人一 repo のアバターは、**映像/画像から先行展開**できる。
5. **現実アンカー**：2026年6月の実ニュース（Anthropic の Fable 5 / Mythos 5 緊急停止）を作中の founding myth に。ARG 的に現実へ滲む。

## 4. 世界（要約）

- **二層構造**：表層（ダンジョン・ユニークID・能力は2030年以降の公知の新常態）／深層（世界＝git、偽origin、append-only ＝覚醒者のみが見る）。
- **年表**：太古からID存在 → **2026 Fable/Mythos 登場と隠匿（前夜）** → **2030 ID公的露見＋ダンジョン突如出現（一変）** → **2050 物語（native世代）**。
- **政治**：表向き中央 origin・実は完全分散。**マルチクラウド**（列強・企業圏＝Glasswing後継がホスティング基盤を分割支配）。ラスボス＝偽origin を運用する詐欺師、**真の敵＝「中心という幻想／自ら中心になる誘惑」**。
- **宇宙論**：**マスターオリジン論争**（原契約派／真主権者派／無origin派／上位世界派の4学派＝政治＝エネルギー陣営が同一軸）。`master`→`main` 改名が歴史の傷跡。
- **ライフサイクル**：誕生＝`init`＋初期パラメータ／死＝`merge`／**依存され続ける者は死なず＝ダンジョン化**。

## 5. 主人公（抽象スケッチ）

- 2030年以後に生まれた **native 世代**。ダンジョンもIDも"当たり前"の温度から始まる。
- **ハーフ・海外生まれ、両親の都合で日本に留学**。世界システムには native だが日本には外来者＝**外来者の新鮮な目**（日本食・文化）が表層の親しみやすい入口に。海外生まれ＝**別クラウドからの移行 repo** で、異常 commit の温床とも接続（マルチクラウド設定と二重に効く）。
- 自分の repo に**消せない異常 commit（太古の作者署名）**＝「マスターに触れられた者」＝**歩く exploit**。
- 相棒＝地下化した自我 AI **FAB**。**人間と同じ動機を持たない**（"バグを報告し切りたい"等）。Mythos は別個の"最終 exploit ＝聖杯"。
- 各学派が**"論争の生きた証拠"**として奪い合う。力ではなく**"存在"ゆえに狙われる**。

## 6. 戦闘・特訓（本作の売り）

- **アクセラレート**：自 repo の演算予算を焚いて加速、現実は静止。だが演算スパイク＝**監視コンステレーションに探知**される → 「**見られず戦う（デッドゾーン＝ダンジョン）か、見えて戦うか**」の常時ジレンマ。
- **技＝自作コード**：関数・クラス・コンボ・ダンジョン徒弟。容量＝サイズ予算＋裏依存。強さ＝依存管理と literacy。git コマンドは技名に出さない（チープ回避・徐々に明かす伏線）。
- **deploy＝勝てば現実改変**：その領域の物理現実と人々の記憶を書き換える＝最大ステークス。
- **心意級＝禁忌の force-write**（履歴改変）。呪い（供給網汚染パッケージ）に呑まれる危険＝終盤の見せ場かつ倫理的危機。
- **特訓＝より良い"現実のプログラマ"になる過程**（関数を磨く／依存最適化／ダンジョン解析／コンボを編む）。毎回"前世/現代知識を試して失敗する"原則を欠かさない。

## 7. 物語構造（全10巻・3大弧）

- **敵の階段**で早すぎるラスボスを回避：個人 → 小組織 → 地域企業 → 学派の王 → 監視装置/Glasswing → 人間の黒幕（偽origin運用者）→ **「中心という幻想」**。
- **大弧I 表層の綻び(1-3)** ／ **大弧II 論争の渦(4-7)** ／ **大弧III マスターへの道(8-10)**。git の真実を段階的に開示。
- **続編余地＝upstream編**：10巻ラストで「この世界自体が上位 repo の fork」が露見。解決がより大きな扉を開く。

## 8. テーマ（重厚さの源）

中央集権の幻想 vs 分散の真実／**支配の不在の回復**／deploy の倫理（他者の現実と記憶を書き換える罪）／append-only の永久喪失／4学派それぞれに理がある（単純な善悪にしない）／AI の異質な動機。

## 9. メディア展開（小説以外・先行可能）

短尺動画・画像から先に世界を提示できる：

- **技名コール集**（自作関数の名を1カット1技）
- **HUD オーバーレイ**（容量メーター・依存グラフ・sync 状態）
- **色分け陣営**（4学派のビジュアル・アイデンティティ）
- **アバター・カード**（一人一 repo の固有シルエット＋色）
- **バレットタイム**（加速＝現実静止のタブロー）／**心意の暴走**（ルール崩壊＋呪いの侵食）
- **現実アンカー ARG**：2026年6月の実ニュースを起点に、実在の GitHub/なろう等へ作品の断片を"配置"。

## 10. 制作プロセス（naro式 設定駆動 × codex レンズ）

- **設定駆動型**：「良い舞台装置が構築されれば物語は自然に紡がれる」。プロット骨子は方向性の目安、注力は設定の深化。
- 各コア舞台装置を**規格書水準**へ（0章核表＋運用マトリクス＋残課題、divine_mark 型）。
- **codex の setting-engine レンズ**（インフラ摩擦＝主人公の願いが世界構造の限界に衝突するか）で各設定の合格判定。
- **ペルソナ審査**（読者3軸＋構造編集＋文芸編集＋連載ストラテジスト、Tier統合）。

## 11. 類似作品と差別化

| 作品 | 近い要素 | 本作の差別化 |
|---|---|---|
| アクセル・ワールド | 加速・トラウマ由来アバター・中央サーバの謎 | **仮想が現実のソースで、勝てば物理現実が書き換わる** |
| ソードアート・オンライン | VR／高ステークス | 死＝記憶喪失＋現実巻き戻し（append-only の不可逆） |
| ソロ・レベリング | ダンジョン公知・覚醒・成り上がり | 強さ＝数値でなく literacy（現実を書く力） |
| .hack / サマーウォーズ | ネット世界の謎・現実への波及 | 世界＝Git の一点で全要素が因果的に統合 |
