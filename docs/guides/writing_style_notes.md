# 文体・用語ノート（v0.1）

> 本文（プロース）・台詞・在野の呼称に適用する文体方針。設定規格書（作者側）とは別レイヤ。
> 関連：試作 [prototype_ep1_battle.md](../manuscript/prototypes/prototype_ep1_battle.md)／**AI執筆の癖チェック**＝[ai_writing_pitfalls.md](ai_writing_pitfalls.md)（機械検査 `scripts/check_manuscript.py`）。

---

## 1. 基本レジスター

- **ライトで読みやすい現代口語**。視点は16歳 native（ハーフ・外来者）。
- 短文・テンポ重視。軽い口調。
- **漢字レベル＝ライトノベル相当**（重要・著者FB 2026-06-14）。**日常語は通常どおり漢字で書く**。「漢字をひらく」のは**難読字・古風な字・専門語だけ**。日常語を不自然にひらがなにすると、かえって幼く・読みにくく・違和感が出る（やりすぎ厳禁）。

### 1.1 漢字レベルの基準（LN相当・著者FB）

| 区分 | 方針 | 例 |
|---|---|---|
| **日常語＝漢字で書く** | 中学生が読める常用漢字レベルは漢字。ひらがなにしない | 後ろ・昨日・昨夜・夕べ・途中・同じ・重なる・十分・全部・少し・本当・分かる・違う・怖い・優しい・一つ・一人・初めて・確かに・一番・真ん中・中身・周り・間・方・温もり・寂しい・続ける・慌てる・伸ばす・触る・偽物・恐れ |
| **ひらがなで書く＝難読・古風・専門語のみ** | 読めない/硬い字、または専門語は平仮名（または平易語に置換） | にじむ（滲む）・つかむ（掴む）・ほどける・たすき（襷）・まとう（纏う）・なであげる（撫で）・こだま・とっさに（咄嗟）・かばう（庇う＝§庇は禁止）・ぞわぞわ等の擬音 |
| **擬音・口語の崩し** | テンポのため平仮名・カタカナ可 | ふっと・すうっと・ぱりんと・ぐちゃぐちゃ・マジか・なんも |

## 2. 避ける

- 古風・文語・難語：例「貪る」「可もなく不可もない」「灼ける」「綴る」「判じる」「〜であった」。
- **硬い造語をそのまま在野名/本文に出す**：例「異常印」。設定上の正確語は作者側に留め、本文・台詞・在野名は平易な言葉にする。

## 3. 好む

- 日常語・カジュアル語：例「食べに来た」「マニュアル」「雑魚」「ありえない速さ」「さわっちゃいけない」。
- 在野の呼称は平易に。例：主人公の印 → 在野では「**あざ**」。

## 4. 維持（既存原則）

- **現象として提示**（機構を地の文で説明しない）。
- **git 語を表に出さない**（作者側の整合エンジンに留める）。
- **技名は自由命名**（自作コードの名）。**主人公（ハーフ・海外育ち）の技＝英語/カタカナ寄り＋雨 motif**（彼に自然・没個性を避ける）。例：守り＝《アンブレラ》（傘＝ひとりぶんの雨よけ。**見え方＝「半透明のシールド」で描く**＝著者FB 2026-06-14）／消す＝《サンブレイク》（雨が切れて光がさす一瞬＝「一回でいいから止んでほしかった」）。「軒下／晴れ間」は和風すぎるため不採用。

## 4.5 分量と密度（あっさり防止）

- **1話の目安＝5,500〜6,500字**（空白除き。ライトのまま密度を出す）。短文・テンポは維持しつつ、薄くしない。
- **密度の四要素**：1シーンに次を意識的に入れる。**①五感**（場の手触り・におい・音・温度）／**②内面**（ナギの心情・ためらい・恐怖）／**③会話**（掛け合いのラリー。説明を会話に逃がす）／**④戦闘の身体感覚**（重さ・痺れ・息）。
- **「ライト＝薄い」にしない**：平易な短文のまま、上の四要素で厚みを出す。
- **専門語・予備知識が要る語を本文に出さない**：例「あふれる（容量超過）」「庇」→ 平易な体感語に（「抱えきれず潰れる」／守り《アンブレラ》の見え方＝「半透明のシールド」）。能力の比喩（「書く＝起きたことに手が届く」）は**初出で効果を具体に見せて**意味を通す。

## 5. before / after 例

| before（硬い） | after（ライト） |
|---|---|
| 菌糸を貪りに来た中型 | 菌糸を食べに来た中型のやつ |
| 規定を裏切る速さで距離を詰めていた | ありえない速さで距離を詰めてくる |
| ひとりぶんだけ濡れずにいられる、それだけの術だ | ひとりぶんだけ濡れずにいられる、それだけの術 |
| 異常印（造語・廃止） | あざ（自然発生）／タトゥー（人工） |
