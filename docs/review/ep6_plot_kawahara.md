# 【総評】マージ可否

**判定：条件付き可（必須修正2件・推奨修正3件＋確認1件）**

第5話で「いまのきみには、別の帰り方が、無い」という**欠落の問い**を残した。その回収の入口として、第6話の設計思想は正しい。第5話のレビューで私が「答え(go dark)は見せず、穴だけ残す」と言った、その穴に**輪郭**を与える――「核に降りて種を持ち帰れば足跡を断てる。だが核は深く、いまは降りられない」。**答えの形は見えるが手は届かない**で止める。これは『アクセル・ワールド』が能力の真価を巻をまたいで小出しにする、あの呼吸とよく揃っている。第3ブロック（第9話・核到達）の payoff を温存する判断も、原則6（コスト）に誠実だ。go dark（遅い・自力・危険）↔ 削れ（速い・FAB・空洞化）の二択を、追跡という具体的な圧の中に立てた構図も骨が良い。

ただし、この回には**構造上の弱点が二つ**ある。一つは、この話の生命線である「**答えが見えるのに手が届かない**」という宙吊りが、**FABの一回の台詞説明だけで成立しようとしている**こと（指摘1）。もう一つは、go dark↔削れの二択が**ナギの内面ロジックとして十分に言語化されておらず、「保留」という結論だけが先に置かれている**こと（指摘2）。この二点は原則4（内面ロジック）と原則2（牽引力）の核心で、本文化前に橋を架けておかないと、「FABが答えを教え、ナギが何もせず保留して終わる回」に倒れる危険がある。残り三点は鵜飼の包囲の説得力・引きの二択・喪失の鏡の溜め方で、いずれも推奨。開示面はおおむね上限を守れている（後述・指摘6）。

## 個別指摘（観察→原因→提案）

### 指摘1：【必須】「答えは見えるが手が届かない」が、FABの一回の説明で立ってしまう

- **観察**：この回の心臓＝go darkの糸口は、B2でFABが「核のいちばん深いところへ降りて種を持ち帰れば足跡を残さず戻れる／ただしそこは危ない、いまのきみには降りられない」と**一息で説明する**形になっている。ログライン・§1・B2のすべてが、この**一回の台詞**に「答えの提示」と「手が届かない理由」の両方を背負わせている。
- **原因**：原則2（牽引力）と原則4（内面ロジック）。第5話で「**穴だけ**残す」のが効いたのは、答えが**見えなかった**からだ。今回は答えに輪郭を与える回なので、宙吊りの質が変わる――「見えているのに届かない」を成立させるには、**読者（とナギ）が「だったらすぐ行けばいい」と思った瞬間に、その手が弾かれる体感**が要る。だがFABが最初から「降りるのは危ない」と**先回りして封じる**と、ナギは試す前に諦めることになり、「届かない」が**台詞の宣言**で終わる。読者は「FABがそう言うなら」と受け取るだけで、**手を伸ばして弾かれた痛み**を共有できない。糸口の輪郭が、希望でなく**説明**として消費される。
- **提案**：「届かない」を**台詞でなく体感で一度立てる**。たとえばB2でFABが種の在りかを示したあと、ナギが**思わず核へ降りる継ぎ目に一歩近づこうとする**――その瞬間に、B6で書かれている「空気が濃い・胸が締めつけられる」**喪失の鏡の圧が、身体で押し返してくる**。FABの「危ない」は、その**後追いの確認**にする（「ね? いまのきみには、まだ早い」）。順序を「①答えが見える→②思わず手が伸びる→③身体が弾かれる→④FABが言語化する」にすると、**手が届かないことを読者がナギと一緒に体で知る**。B6を末尾に置いたまま「核の入口に立つ」を一度きりにするより、**B2で軽く触れて弾かれ、B6で本格的に溜める**二段にした方が、糸口の宙吊りが回を通して効く。これが無いと、この回の核が「FABの説明回」に痩せる。

### 指摘2：【必須】go dark↔削れの二択が、ナギの内面ロジックでなく「保留」という結論から書かれている

- **観察**：§1・B3・B7・残課題のすべてが、二択を「**速くて楽な道（削れ）↔ 遅くて危ない自力（核へ降りる）**」と整理し、結論を「**天秤は未決のまま**」「ナギは『まだ』とだけ答える」に置いている。B7の内面は「速さか、深いところか。どちらも怖い。どちらも、何かを払う」と**二択の構図を要約する**形。
- **原因**：原則4（内面ロジックの密度）。川原礫作品で「この主人公、ちゃんと考えてる」と読者が感じるのは、主人公が**限界まで頭を使って、それでも決めきれない**プロセスが見えるからだ。今のプロットは「二択がある／どちらも怖い／だから保留」と、**選択肢の存在と感情（怖い）だけ**で、**ナギが各々を本気で天秤にかけて量る思考**が抜けている。「保留」は**結論**だが、結論だけ置くと、読者には「決断を先送りした回」に見える。第4話「受ける」（削れを受諾）→第6話「保留」の天秤の前進（skeleton §1.6）を効かせるには、**保留が"逃げ"でなく"どちらも選べないほど両方が重い"という能動的な宙吊り**だと読者に伝わらないといけない。
- **提案**：B3かB7で、ナギに**二つを具体的に量らせる一拍**を入れる。削れ側＝「今夜整えれば、明日には振り切れる。でも――おばさんの一言が消えたみたいに、また何かが消える」（第4話の代償の**具体的な記憶**を引いて天秤に乗せる）。go dark側＝「核に降りれば足跡を断てる。でも、あそこは見ているだけで胸が潰れる。降りて、無事でいられる気がしない」。**両方の皿に、抽象でなく具体の重りを乗せる**。そのうえで「だから、まだ決められない」に着地すれば、保留が**思考の果ての宙吊り**になる。FABが二択を**提示する**のでなく、ナギが**自分で二択を組み立てて立ち尽くす**形にすると、原則7（キャラ主導）も同時に立つ。第5話の私の指摘3と同じ思想だ――答えの存在を匂わせつつ、**選べなさの内実**を書く。

### 指摘3：【推奨】鵜飼の包囲＝「点vs面でやり過ごすが根本解決でない」の説得力

- **観察**：B4でナギが面で「鵜の見ている筋」を読み、澪が囮の一射で逸らす＝守りの連携。B5で「やり過ごしても、坂を下りきるとまた新しい光が灯る＝一時しのぎ」。設定影響表でも「点vs面でやり過ごせるが根本解決でない＝go darkが要る理由を機構で示す」と明記。
- **原因**：原則4（内面ロジック）・原則6（コスト）。**思想は完全に正しい**。だが「やり過ごす→また増える」を**説得力のある必然**にするには、「なぜ逸らしても無駄か」が読者に**機構として腑に落ちる**必要がある。第5話で「速さは足跡を濃くするだけ」「鵜は新鮮な足跡しか追えないが、通うたび新しい足跡が出る」という**機構の理由**を立てた。今回の「囮で逸らしてもまた灯る」も、同じ機構の延長で説明できるはず――**囮は今ある目を一つ逸らすだけで、ナギが隠れ家に通う限り、新しい足跡＝新しい目が湧き続ける**。この「**逸らす（点対処）では、湧き続ける源（=通う足跡そのもの）を断てない**」という因果が、B5でナギの認識として一拍言語化されると、「だからgo dark（足跡の源を断つ）が要る」が**論理として**繋がる。今のプロットはこの因果が「また灯る」という現象の提示で止まっており、ナギが**なぜ根本解決にならないかを理解する一拍**が薄い。
- **提案**：B5でナギに「**逸らしたって、明日もここに帰ってくる。帰る限り、また足跡が増える。一つ消しても、源は自分だ**」と気づかせる。**点（囮＝個別対処）では面（通い続ける構造）を断てない**――この認識が、第2話以来の「点vs面」の主題を**今度はナギ自身の問題**に折り返す。そうすればB2で示された糸口（核に降りて足跡の源を断つ＝go dark）が「**唯一、源を断てる手段**」として読者の中で必然化する。「やり過ごす」が**敗北でなく、根本解決の必要性を体で知る通過点**になる。

### 指摘4：【推奨】核＝喪失の鏡を「降りずに溜める」判断は正しいが、降りない理由を内面で支えたい

- **観察**：B6でナギは核へ降りる継ぎ目の前に立ち、「空気が濃い・待ち続けて発てなかった誰かの気配・胸が締めつけられる」を感じ、**今日は降りない**（怖さの溜め・本番は第9話）。FABの「今日は、やめておく? 無理は、しなくていい」が降りない判断を後押しする。
- **原因**：原則6（コスト）・原則4（内面）。**「降りずに溜める」判断そのものは正しい**――第9話の核到達 payoff を温存するため、ここで降ろしてはいけない。問題は**降りない理由がFABの気遣いに乗る形**になっていること。指摘2と同根で、ナギが**自分の意志で「まだ降りない」を選ぶ**のでなく、FABに「やめておく?」と言われて頷くと、**喪失の鏡の怖さ**（この回で一番静かに刺さるべき情緒）が、ナギの能動的な恐怖でなく**外からの許可**で処理されてしまう。
- **提案**：降りない判断を**ナギの側に置く**。FABが「無理はしなくていい」と言う**前に**、ナギが継ぎ目の奥を見て、**自分から後ずさる**一拍を入れる。「降りれば種が手に入る。分かってる。でも――足が、動かない」。降りられないのが**怖さ（喪失の鏡）に身体が拒否反応を起こす**からだと身体で示せば、第9話で「それでも降りる」決意が**重さを持って**返ってくる。喪失真相は伏せたまま（disclosure準拠）、「**なぜここがこんなに怖いのか、自分でも分からない**」という**理由の分からない圧**にしておくと、鏡像の不気味さ（第5話で澪の既視感に使った"理由が分からないからこそ怖い"の作法）と揃う。これは原則4の「なぜそう判断したか」を、喪失の縦糸で支える手だ。

### 指摘5：【推奨・確認】引きの二択は性質が違う。回の重心がどちらに傾くかを意識して選ぶ

- **観察**：B8の引きは①核へ降りる継ぎ目の奥から「誰かがこちらを見上げた」気配（喪失の鏡・第9話の核到達への引力）②入口の白い光が夜のあいだに二つに増えている（鵜飼の包囲の進行）、の二択。残課題で本文化時・著者確認送り。
- **原因**：原則5（引き）。**どちらも一撃に絞れており、引きの作法自体は良い**。ただし二つは**引っ張る縦糸が違う**。①は**喪失・核（go darkの目的地）**へ引く＝この回が「帰り方＝核へ降りる物語」だと読者に印象づける。②は**監視・包囲（鵜飼の圧）**へ引く＝この回が「追われる物語」だと印象づける。どちらを採るかで、**第6話が何の回だったか**の読者の記憶が変わる。
- **提案**：この回の主筋は**go darkの糸口＝核**だ（タイトル「帰り方」もそこを指す）。なら引きは①（核の気配）の方が、**回の主題と引きが一致**して強い。②は鵜飼の圧を引きにすると第5話（入口の白い光）と**型が重なる**（同じ「白い光が残る/増える」の引き）。①の「核の奥から誰かが見上げた」は、**この回で初めて立てた核**を引きに使う新規性があり、第9話への引力も直接的だ。私は①を推す。ただし**①は喪失真相に踏み込みすぎないか**だけ本文化時に確認したい（指摘6）。②を残すなら、第5話との型の重複を避ける見せ方（光の数でなく、たとえば「**昨日まで無かった、隠れ家の継ぎ目"そのもの"の手前に一つ**」＝中まで詰められた質の変化）にずらすと差別化できる。

### 指摘6：開示（disclosure）面の構造チェック

- **観察**：プロット全体で、go dark／シードは「核／いちばん深いところ／種／降りて持ち帰る」の体感語に、喪失真相は「待ち続けて発てなかった者の気配」の気配のみに、削れは現象語（あざの熱）に、鵜飼の過去は不出に、それぞれ置き換えられている。git/seed語・"上"も出していない。
- **原因**：B1開示上限（disclosure §0/§2、skeleton §0 隠蔽上限）。**基本は上限を守れている**。シード＝核到達で取得（dungeon §1.5-6）を「種を持ち帰る」の体感語に翻訳する方針は正しく、git/seed語不出も徹底されている。喪失真相＝気配のみ、鵜飼の過去＝不出、も保たれている。
- **提案**：二点だけ本文化時に注意したい。**(a) B2/B8の「核」「種」「いちばん深いところ」の語彙**＝残課題にも挙がっているが、「種」は seed の直訳に近いので、**読者が"これはシステム用語だ"と気づく一歩手前で止める**言い回しの精査が要る（「種」より「いちばん奥にあるもの」「核の芯」等、体感に寄せる候補も並べて disclosure 照合を）。**(b) 指摘5の引き①「誰かがこちらを見上げた」**＝喪失の鏡を気配で立てるのは良いが、「**待ち続けて発てなかった"誰か"が、ナギを見て反応した**」と踏み込むと、その"誰か"とナギの関係（喪失真相）に読者の認知が一歩近づく危険がある。「**見上げた**」までで止め、**それが誰か・なぜナギを見たかは一切出さない**（FABも解説しない）処理を徹底したい。第5話で鵜飼を「自分も理由を知らない手」に留めたのと同じ、**気配の手前で踏みとどまる**設計だ。

## 全体の懸念点と提案（原則2・4・6を主軸に）

この回の生命線は、**「答えが見えるのに手が届かない」を、説明でなく体感で張り続けられるか**だ。第5話は答えが見えなかったから穴だけで成立した。第6話は答えに輪郭を与える回なので、**輪郭を見せた瞬間に手を弾く**機構が要る。FABの「危ない」という台詞でなく、**ナギが思わず手を伸ばして、喪失の鏡の圧に身体で押し返される**――この一連を立てれば、糸口が希望として灯り、同時に届かなさが痛みとして残る（指摘1）。

そのうえで、go dark↔削れの二択を**ナギの思考として書く**（指摘2）。両方の皿に具体の重りを乗せ、「だから、まだ決められない」を**選べなさの内実**として届ける。保留が逃げに見えないために。

残りは説得力の補強だ。鵜飼の包囲を「点では源を断てない」の因果で論理化し（指摘3）、核を降りない判断をナギの能動に置き（指摘4）、引きは回の主題と一致する核の気配を推す（指摘5）。開示は概ね守れているが、「種」の語彙と引きの踏み込みだけ本文化時に照合したい（指摘6）。骨は良い。**答えを見せて、その手を弾く。二択を量って、決めきれない。** この二つの宙吊りを体感で立てれば、第2ブロックの"張り続ける"5話の二本目として、第9話の核到達へきれいに引っ張れる回になる。

第5話で私は「答えは見せず、穴だけ残す」と言った。第6話は、その穴に**手を伸ばさせて、届かせない**回だ。難しいのは、答えを見せた瞬間に読者が「じゃあ行けばいい」と思うことで――そこで**身体ごと弾かれる**痛みを一度通すと、「届かない」が宣言でなく体感になる。go darkも、削れも、どちらも何かを払う。**払えないから、決められない。** その立ち尽くしを、感情でなく思考で書ければ、この回は「保留の回」でなく「**二つの怖さの間で、限界まで考えて、それでも動けない回**」になる。骨は良いね。あとは、宙吊りを台詞から身体へ引き下ろすこと。

# 必須／推奨と収束ポイント

**判定：条件付き可（必須修正2件・推奨修正3件＋確認1件）**

### 最優先の収束ポイント（3点以内）

1. **【必須】「答えは見えるが手が届かない」を、FABの説明でなく体感で立てる**（原則2・4／指摘1）。B2で答えの輪郭を見せたあと、ナギが思わず核へ一歩近づき→喪失の鏡の圧に**身体で押し返される**→FABが後追いで言語化、の順にする。B2で軽く弾かれ・B6で本格的に溜める二段に。これが無いと回の核が「FABの説明回」に痩せる。

2. **【必須】go dark↔削れの二択を、ナギの内面ロジックとして書く**（原則4・7／指摘2）。両方の皿に**具体の重り**を乗せる（削れ＝消えた口ぐせの記憶／go dark＝降りて無事でいられない予感）。「保留」を結論でなく、**限界まで量った末の選べなさ**として届ける。第4話「受ける」→第6話「保留」の前進が、逃げでなく宙吊りに見えるように。

3. **【推奨】鵜飼の包囲＝根本解決でない理由を「点では源を断てない」で論理化／引きは核の気配を推す**（原則4・6・5／指摘3・5）。B5で「逸らしても、帰る限り足跡の源は自分」とナギに気づかせ、go dark（源を断つ）の必然性を論理で繋ぐ。引きは回の主題（核＝帰り方）と一致する①「核の奥から見上げた気配」を推す（②は第5話と型が重複・残すなら見せ方をずらす）。喪失真相への踏み込みは disclosure 照合（指摘6）。
