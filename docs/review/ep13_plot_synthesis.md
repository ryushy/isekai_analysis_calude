## 1. 反映する必須

### A.【必須・全員＝最重要＝villain化回避】澪の踏み越えを"観念の独白"でなく"震える指（身体）"で立て、境界はナギが「名指して確かにする」
- 問題＝ビート2で澪が「鵜飼は誰のためでもなく／わたしはこの人のために／一点だけが違う」と**境界を整然と自己説明**＝軍師口調になり①澪の芯が"正義の決断"に滑る（villain化／[heroine §0.5](../settings/heroine_core.md)「間で語る・短く言い切る」から浮く）②芯の確立がビート6のナギ頼みになり主導性を肩代わり（九条）。
- 反映＝**①ビート2の境界の言語化（整然の独白）を削り、澪は揺れたまま＝答えの出ない一拍・震える指で示す**（夏目・霧島）。**②ビート3の差し込む直前に、澪自身の動機（この人のために怒る／守りたい）を身体で立てる＝villain化回避の生命線＝澪の芯は澪自身のもの（ナギが供給しない）**（九条・読者）。**③ビート6のナギの境界固定を"論理"でなく"澪の身体を名指す"へ＝「お前の手、震えてる。あいつの手は、震えねえよ」（証人の観察）**（霧島・久遠）。＝動機は澪の身体、言語化（確かにする）はナギ、の役割分担で両立。

### B.【必須・九条/読者/久遠＝因果の念押し】澪一人の偽証がなぜ効くかを一片立てる
- 問題＝澪一人の偽報告で鵜飼の手続きが空転する因果が、スケール差（末端一人 vs 機構）で疑われうる／第11話の勝利条件の起動が天下りに見えうる。
- 反映＝**「観測（鵜の目）と手続き（裏取り）は別の層／澪が"確認の欄"＝急所に立っている」を一片仕込み**、ビート4の空転を機構的に成立（[antagonist §2](../settings/antagonist_ukai_core.md)＝抹消申請は面側の裏取りが要る、の起動として。後出しでなく仕込みの回収に）。

### C.【必須・久遠/読者/夏目＝三つ子化回避＝陰りの質を峻別】圧の転換と引きを"質・個人化"で撮る
- 問題＝「勝ち→陰り」の締めリズムが三度。第11（傾く＝向き）・第12（増える＝量）の"面が動く"運動を本話もなぞると三つ子化。
- 反映＝**①ビート5＝白い目が"一つだけ"澪へ向く＝面を動かさず一つの目が澪へ向き直る（質・個人化／向き・量の"多"から降りる）**（久遠）。**②引き＝"澪の外側に灯る、ひとつの目"の外形の画に絞る（第12話＝澪の内面に決意／本話＝澪の外側に監視の目＝内と外で撃ち分け）**（夏目・久遠）＝第12話の引きとの双子化回避。

### D.【必須・霧島/夏目＝鵜飼の無感保持】ビート5の鵜飼を"予言する悪役"から"観測所見を述べる無感"へ
- 問題＝「似ています／いずれ」の台詞が強く、無感を所作で保証しないと"勝ち誇る悪役"に堕ち、澪の恐怖が"脅された"レベルに薄まる（澪が本当に怖いのは脅しでなく**無感に同類と見られること**＝[antagonist §1.5](../settings/antagonist_ukai_core.md)）。
- 反映＝**「似ています」を平坦・間なし・言い捨て・反応を待たないで言わせる**（脅すより見限る方が怖い）。予言の強度は台詞を弱め所作・無感へ。

### E.【必須・九条＝視点の特例明示と戻し導線】澪寄りは"本話の特例"と位置づけ、第14話でナギ寄りに戻す導線を確定
- 問題＝ナギ寄り規律を本話だけ破る代償（流し読みの一瞬の戸惑い・第14話の戻し判断）。
- 反映＝**§5決定欄に「本話＝澪寄り（特例）／第14話＝ナギ寄りに戻す（ナギ視点で踏み越えた澪の背から入る＝本話ビート8別案が第14話の入口）」と戻し先まで確定**（視点を動かすなら戻す先まで決めてから動かす）。

### F.【必須・夏目＝緩の谷】四話連続で緩がゼロ＝呼吸を一拍入れる
- 問題＝8ビート全部が低温か緊張で、溜飲（ビート4）が陰り（ビート5）に即隣接して呼吸できない（第12話の宿題の再発）。
- 反映＝**溜飲後（ビート4）に"効いた"を噛む素の温度の半拍＋冒頭（ビート1）にいつもの夜の温もりの谷を置く**。

## 2. 反映する推奨／確定

- **溜飲を澪の身体で締める**（九条・霧島）＝ビート4を鵜飼側の現象でなく、震えた澪の指が効いてほどける身体の像で（安堵と痛みの同時＝止めた息を吐く→守った手／礼を破った手を見る）。言葉にするのはナギ、身体で受けるのは澪。
- **弓を引く指の身体記憶の対**（霧島・久遠）＝偽証する指が、ふと弓の構えを思い出す＝身体記憶の矛盾が震えの理由（震える手 vs 震えない手の境界をここで半分仕込む／「見てから放つ」の残心の破れた版＝震え→息を止める→差し込む の三拍）。
- **澪の口調を戻す**（夏目）＝踏み越えの決意台詞を、です/ます・短く言い切る・素の戸惑い「たぶん」に削る（軍師口調＝鵜飼との近接を避ける）。
- **引きの三段の積み**（霧島）＝ビート5（首のぞわり）→ビート7（予兆）→引き（確定）で「見る側→見られる側」の身体記憶の反転を積む（引きで初めて反転を出すと唐突）。
- **ビート1の機構再説明を削り所作に絞る**（霧島）＝弓巻→端末の手の移動で（第11話の告白の反復回避／「見られる側になります」の先植えは残す）。**入り口を"次は、わたしの番"のpayoffとして渡し目で丁寧に開く**（読者）。

## 3. 見送り/次話・著者決定

- **残る著者決定＝タイトル**（内から／裏取り／見られる側／似てくる）**／引きの視点**（澪寄り＝5者支持・採否確認・引きは"澪の外側に灯る一つの目"の外形）。**既視感セリフの強度は収束Dで確定**（予言・無感まで・来歴は語らせない）。
- **視点の戻し**：第14話「渡さない」＝ナギ寄りに戻す（本話ビート8別案＝ナギ視点で踏み越えた澪の背から入る、が第14話の入口）。
- **決定的喪失（澪が登録の側を失う）は巻2へ温存**（§1.7）。本話は予兆のみ（澪に向く目＝一つまで）。
- **削れ／go dark＝本話では出さない**。第14話「渡さない」の最終拒絶に温存（第14話プロットで整合確認）。
