# 【総評】マージ可否

**判定：条件付き可（必須修正1件・推奨修正3件）**

第2ブロックを起動する設計として、骨は良い。私がこのプロジェクトで一番信頼している背骨――**原則6（コスト）**を、今回は「第4話で澪を入れた＝合流の足跡が脆弱性を生んだ」という形で、**得た力（速さ・仲間）が解決にならず、むしろ追跡を呼ぶ**皮肉に変換している。インフレ回避の思想が一貫している。鵜＝白い光の観測ノードという**静の脅威の意匠**も、道化（無秩序）↔鵜飼（秩序の冷たさ）の対称として機能する。引きを「入口の白い光ひとつ」に絞った判断も第2〜4話の作法と揃っている。

ただし一点、この回の**心臓の因果**――「**澪の合流が新鮮な足跡を残した**」という皮肉が、**読者の頭の中で因果として結べるか**に、構造上の穴がある。ここは設定書（dungeon §1.5-6・ukai §2）では完璧に閉じているが、**B1開示上限の縛りで本文には「git/合流＝共有アンカー」の語が出せない**。つまり「なぜ今・なぜここで追われ始めたか」を、読者は**機構語抜きで体感しなければならない**。プロット段階でこの橋渡しの仕込みが弱いと、本文化したとき「敵が都合よく現れた」に倒れる。これが唯一の必須修正だ。残り三点は、引きの別案・シードの予感処理・密度の整理で、いずれも推奨。

## 個別指摘（観察→原因→提案）

### 指摘1：【必須】「澪の合流が足跡を残した」皮肉の因果が、読者の頭で結べるか

- **観察**：ログライン・§1・B2・B4・設定影響表のすべてが、この回の核を「**第4話で澪が合流（共有アンカー＝戻り標）した新鮮な足跡を鵜飼が読む＝澪の関与が脆弱性を生んだ皮肉**」に置いている。設計書（dungeon §1.5-6・ukai §2）ではこの因果は完璧に閉じている。だがB1開示上限で、本文には「合流＝共有アンカー」「戻り標」「git/sync」の機構語が**出せない**（disclosure §0/§3）。
- **原因**：原則4（内面ロジック）と原則7（キャラの動きから世界が立ち上がる）の境界。皮肉が刺さる条件は「**A（澪を入れた）→B（だから追われ始めた）**」を読者が**自分で結ぶ**こと。だが機構語抜きだと、B2で白い光が「昨日まで澪と合流に使った道」に出る、という**配置**だけが頼りになる。プロットはこの配置を持っているが、**「昨日まで誰もいなかった道」＝「澪と通った道」だと読者が認識し直す一拍**が、ビートに明示されていない。これが無いと、読者は「敵が（理由は分からないが）現れた」と受け取り、皮肉が「都合のいい登場」に化ける。第4話で「戻り標を合流ルートに転用した継ぎ目」を私は秀逸だと評価したが、その継ぎ目の**回収**が今回の生命線だ。
- **提案**：B2かB4で、**ナギ（またはFAB）が「昨日、ここを澪と通った」と一度はっきり意識する内面の一拍**を入れる。「白い光は、ひとりで歩いてた頃の道には無かった。澪と――二人で通った道にだけ、留まっている」のように、**読者がA→Bを結べる手がかりを、機構語でなく"場所の対応"で置く**。さらにFABの台詞を「足跡が、増えた。……ひとりじゃ、こんなに濃くなかったのにね」程度に**踏み込ませる**と、「仲間を入れたこと」と「読まれること」が読者の中で接続する。ここまでやって初めて、§1の「澪の優しさが冷たい手を引き寄せた皮肉」が**痛みとして**届く。設定書の因果を、本文の体感に翻訳する橋を、プロットに明記しておきたい。

### 指摘2：【推奨】B3「報告照会」が外から来るため、澪の軋みが受け身に倒れる

- **観察**：B3で澪の軋みは「**上（監視網）からの確認照会**＝『未報告の潜りがいるはずだ。心当たりは』」という**外発のトリガー**で起動する。残課題でも「誰から・どんな形で来るか」は本文化時送りになっている。
- **原因**：原則7（キャラの主導性）。「保留が露見し始める圧」は縦糸として正しい。だが**照会が外から降ってくる**形だと、澪は「問われて濁す」リアクション側に立つ。第2話「今日は（見逃す）」→第4話「自分の意思で保留」と、澪は**能動的に**保留を選んできた。その前進線の延長なら、第5話の軋みも**澪の内側から**始めたい。
- **提案**：照会の前に、**澪自身が「自分の報告書に空欄が積み上がっている」ことに先に気づいてぞっとする一拍**を置く。外から問われて初めて気づくのでなく、**自分の手順（手順の人＝heroine §1.5①）が、自分の沈黙を一番よく知っている**――その自己観測から始める。そのうえで照会が来ると、「来てしまった」の恐怖が二段で効く。これは引きの別案「白紙の報告欄の前で指を止める」（B8別案）とも一本に繋がり、澪視点で締める場合の布石になる。澪を**証人かつ当事者**として能動側に置きたい。

### 指摘3：【推奨】シード機構（go dark）の「予感だけ」処理が、まだ言い過ぎ寄り

- **観察**：B4でFABが「**足跡を残すのをやめれば、消える。……だが、それには別の戻り方（＝シード）が要る**」と言う。冒頭メモ・設定影響表でも「答えの予感だけ・第3ブロックへ温存」と明記されている。
- **原因**：原則2（牽引力）と開示設計の境界。**意図は完全に正しい**――答え（go dark）を今出すと、第3ブロックの決着の payoff が痩せる。だが、この台詞は「やめれば消える／別の戻り方が要る」と、**解決の輪郭（手段が存在し、それを取れば追跡を断てる）まで言ってしまっている**。「予感」は**問いの形**で残すのが強い。手段の存在まで明かすと、読者は「じゃあそれをやればいい」と**答えを先に持ってしまい**、第3ブロックで「知っていたことをやるだけ」になりかねない。
- **提案**：FABの台詞を**手段の輪郭でなく、欠落の感触**に寄せる。「足跡さえ残さなければ、この目は君を見失う。……でも、いまの君には、その帰り方がない」程度に留め、**「別の戻り方がある」とは言い切らない**。あるいはFABが**言いかけて黙る**（fab §3「選んで言う／黙る」の作法）。「足跡を残さない帰り方が……いや、まだ君には早い」と引っ込めれば、**答えの存在を匂わせつつ中身は空白**になり、第3ブロックの「核に到達してシードを読む（dungeon §1.5-6）」という**到達の payoff**が温存される。予感は「答えがある気がする」でなく「**ここに穴がある**」で置くのが、川原礫的な引きだ（『アクセル・ワールド』が能力の真価を巻をまたいで小出しにする、あの呼吸）。

### 指摘4：【推奨】B4-B5の「力で押せない相手」が二度提示され、密度が重複している

- **観察**：B4で「削れで得た速さで逃げても速さは足跡を濃くするだけ＝力が解決にならない」、B5で「澪の弓＝点は無数・実体の薄い鵜に通じない／ナギの面でも掴みどころがない＝二人の力の隙間を突く」。**「既存の力が効かない」という同種のメッセージ**が連続する。
- **原因**：原則1（平易性）・原則2（牽引力）。どちらも良いビートだが、**同じ"無力の確認"が二連打**になると、中盤のテンポがそこで足踏みする。読者は「効かないのは分かった」と一度で受け取る。
- **提案**：役割を**分離して一回ずつ**にする。B4は「**逃げる＝動く**ことが裏目（速さ＝足跡）」＝**ナギ単独の無力**に絞る。B5は「**撃つ＝面vs点**が通じない」＝**二人の連携の無力**に絞る。同じ「無力」でも、B4＝個の手詰まり／B5＝連携の手詰まり、と**位相を変える**と重複が解け、「単独でも二人でも届かない」という**段階的な絶望の上げ**になる。FABの台詞もB4とB5で機能を分け（B4＝足跡の濃さ／B5＝帰り道そのものを読まれている）、片方ずつに置けば一文ずつ軽くなる。

### 指摘5：【推奨・確認】B6の顔出し＝「手続き」の通告が、澪の手順を見透かす一言と二重になり密度過多

- **観察**：B6で鵜飼は①名乗り「足取りの鵜飼」②通告「これは処罰ではありません。手続きです」「あなたを消すのは悪意ではありません。いちばん穏やかな結末です」③澪へ「あなたも、登録の側でしょう。……確かめてから報告する。律儀だ」――と、**三つの機能（名乗り・通告・鏡像の種）を一場面に詰めている**。
- **原因**：原則5（引き）・原則4（内面）。鵜飼初顔出しは第2ブロックの大きな結節点で、**情報を盛りたくなる**のは分かる。だが③（澪の手順を見透かす＝鏡像の種）は、B7の既視感と**機能が近い**。B6で鵜飼が澪を名指しで分析すると、B7の「理由の分からない既視感」が**先に説明されてしまい**、既視感の不気味さ（=理由が分からないからこそ怖い）が薄まる。
- **提案**：B6では鵜飼を**ナギに対する通告**に集中させ、澪へは**名指しの分析でなく、ただ一瞥を向けて去る**だけに留める。③の「確かめてから報告する。律儀だ」は**強い手だが、ここで使うと早い**。澪を見透かす一言は、第6話以降の二度目の接触まで温存し、第5話のB7は「**鵜飼が澪に何も言わないのに、澪だけがぞっとする**」――この**非対称**で既視感を立てる方が、ukai §0.5「相手の名を呼ばず『対象』『あなた』で済ます」「わずかな既視感だけを覗かせる」の設計とも整合し、disclosure（既視感のみ）も守れる。鵜飼が**澪を個として認識しない**まま、澪だけが鏡を見てしまう――その一方通行が、この回の「核・鏡像の種」を一番静かに刺す。

### 指摘6：開示（disclosure）面の構造チェック

- **観察**：プロット全体で、git/sync/revert・"上"の正体・鵜飼の過去・道化の正体は、いずれも体感語（「辿られる」「記録を消される」「手続き」「鵜＝白い光・目」）と気配に置き換えられている。FABの「あの道化とは違う／こっちは世界の側だ」も**断定でなく訝しむ**形で、道化（C-028＝気配のみ）と鵜（felt wrongness）の質の差を出すに留めている。
- **原因**：B1開示上限（disclosure §0/§2/§3）。**基本は上限を守れている**。鵜飼C-035＝既視感のみ、道化C-028＝気配のみ、"上"＝伏せる、の三線が保たれている。
- **提案**：一点だけ注意。B6の名乗り「足取りの鵜飼」は名を出すが、**「誰の差し金か」を本人が語らない**処理（末端ゆえ知らない）は正しい。ただし通告「これは手続きです」が**手続きの背後にある制度（＝"上"）の存在**を匂わせすぎないか、本文化時に確認したい。「手続き」「申請」は許容語（ukai §4）だが、**鵜飼が"上"を背負った代理人に見えすぎる**と、読者の認知が制度の輪郭に一歩近づく。鵜飼は「**自分も理由を知らないまま動く手**」に見せる方が、B1の伏せと、後の議長センへの予告編としての"末端の冷たさ"の両方に効く。指摘5で③を引いておけば、ここも自然に軽くなる。

## 全体の懸念点と提案（原則2・4・7を主軸に）

この回の生命線は、**設定書で完璧に閉じている因果を、開示上限の縛りの中で本文の体感に翻訳できるか**だ。鵜飼の機構（戻り標→トレース→新鮮な足跡）は美しく組まれている。だが読者が手にできるのは「白い光が増える」という現象だけ。**現象の配置（澪と通った道にだけ光が出る）と、ナギの認識の一拍**――この二つで「仲間を入れたから読まれた」を読者に結ばせる。これが指摘1（必須）。

残りは温度の調整だ。澪の軋みを能動側に（指摘2）、シードの答えを問いの形に（指摘3）、中盤の無力を位相分けで一回ずつに（指摘4）、鵜飼の顔出しをナギへの通告に絞り澪へは一瞥だけに（指摘5・6）。骨は良い。皮肉の橋と、引きの静けさを整えれば、第2ブロックの開幕として手放せない回になる。

# 必須／推奨と収束ポイント

**判定：条件付き可（必須修正1件・推奨修正3件＋確認2件）**

### 最優先の収束ポイント（3点以内）

1. **【必須】「澪の合流が新鮮な足跡を残した」皮肉の橋を本文の体感に架ける**（原則4・7／指摘1）。機構語を使わず、「澪と通った道にだけ白い光が留まる」という**場所の対応**と、ナギが「ひとりじゃこんなに濃くなかった」と気づく**認識の一拍**を、B2/B4に明記する。これが無いと核の皮肉が「都合のいい敵の登場」に化ける。

2. **【推奨】シードgo darkの予感を「手段の輪郭」から「欠落の問い」に引く**（原則2／指摘3）。FABに「別の戻り方が要る」と言わせず、「いまの君には、その帰り方がない」と**穴だけ**残す。第3ブロックの到達payoffを温存する。

3. **【推奨】鵜飼の顔出し（B6）を整理し、既視感（B7）を"非対称"で立てる**（原則4・5／指摘5・6）。B6で鵜飼が澪を名指し分析（「律儀だ」）するのは早い。鵜飼はナギへの通告に集中し澪へは一瞥のみ、澪だけがぞっとする一方通行で鏡像を刺す。disclosure（既視感のみ）とも整合。
