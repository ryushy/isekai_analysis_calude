# review/ — レビュー結果

> レビュースキルの出力保存先。原稿を「故障モード別」に多角検証した記録。

## 命名

`<対象>_<種別>.md`（例：`prototype_ep1_battle_personas.md`／`prototype_ep1_battle_kujo.md`）。

## レビュースキル（[../../.claude/skills/](../../.claude/skills/)）

| スキル | 観点 | 担当する故障モード |
|---|---|---|
| `reader-personas` | **読者目線**（流し読み・考察・古参の3類型） | 離脱／伏線の粗・整合性／キャラブレ |
| `editor-kawahara`（九条透・川原礫七原則） | **編集者目線**（シリアス寄り構造） | 平易性・牽引力・内面ロジック・引き・ご都合主義 |
| `editor-tsukikage`（夏目凛・槻影七則） | **編集者目線**（コメディ・群像・テンポ） | 会話テンポ・副キャラ判別・連投リズム |

## 運用

- 本文確定後（または試作レビュー時）に走らせ、指摘を改稿に反映。
- 考察勢レビューは [../series/disclosure_ledger.md](../series/disclosure_ledger.md) と照合し「まだ知らないはずを語っていないか」を検出。
