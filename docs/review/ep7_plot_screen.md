## 総評（撮れるか）

**マージ可否＝可（※映像メモは任意・覚え書き扱い）。** プロットとしては撮れる。むしろこの話は「映像化設計」が**プロットの段階ですでに八割描けている**稀なケースで、§3「映像化の見どころ」が決め画を5つ先に挙げている。撮影監督として一番うれしいのは**ビート4の運動の質の転換が言語化済み**（第6話＝守り＝横へ逸らす／第7話＝攻め＝面が形を与え点が射抜く）であること。ここが画になる芯。

ただし**最大の「画にならない空白」が一点**＝**ビート4の"面"の運動が、まだ画の言葉になっていない**。「ナギが滲みの継ぎ目を掴む」が、**何を掴むのか・どう手が動くのか・掴んだ瞬間に画面の何が止まるのか**が未定義のまま。澪の"点"（弓・一直線・残心）は弓道モチーフで誰でも画が立つが、ナギの"面"は**抽象が画に落ちていない**＝ここが本話の戦闘の心臓なのに空白。**§5残課題でも「混沌に形を与える＝面の解像度」を screen-review 送りにしている**＝著者も自覚済み。本レビューの核はここに割く。

開示は厳守できる設計（道化＝署名の物理痕＝火の粉/音/森化のみ・骸骨化粧の"人型"は映さない・FAB＝光のみ・git語不出）。問題なし。

**本文必須：1件**（ビート4の"面"の身体動作＝何を掴むか、最低一行の体の動きが無いと地の文が空白になる）。**推奨：2件**（ビート3の「音だけ先行」のタイミング設計／ビート5の"何も映らない画"の引きの距離）。**映像メモ（任意・絵コンテ送り）：随所**。

## ビート別・キーフレーム設計

### 【ビート1】追われる日常＋宿題への入口
- **観察**：放課後の山手の坂→本牧への動線。白い目の気配がうっすら。澪が「記録の穴」を持ってくる＝「ここ、何かが出たはずなのに、記録が、ない」。
- **見立て（問い1・3・7）**：ここは"日常"のトーンを確立する導入。決め画は**坂の下りの動線＝二人の並走**で立つ。ただし「白い目の気配」を**画のどこに置くか**が未定（端に置かないと後半の並走と連続しない＝問い8）。
- **提案**：
  - **絵コンテに回すなら**：①決め画＝**夕方の山手の坂を下る二人の引き**（縦9:16＝坂が縦に効く・横浜の海光が奥に一筋）。②カメラ＝二人を追うフォロー（背中→横顔へ回り込み）。③光＝マジックアワーの暖色（後半の深層の無機質と温度差を作る伏線）。④**白い光を画面の最奥・港側に一点だけ点す**（鵜飼の気配＝この話を貫く"端の白"の初出。ビート7・引きまで連続させる）。
  - **本文に一行足すなら**：澪が記録の穴を出す時、**手元の端末でなく"視線が一点で止まる"所作**（弓引きの残心＝澪の身体語彙）にすると、説明が画になる。

### 【ビート2】別の場所へダイブ（足跡のリスクを呑んで）
- **観察**：共有アンカー（戻り標）で合流ダイブ。「これで、また、足跡が残ります」／「分かってる。でも、確かめないと」。
- **見立て（問い5・6・移動を画に）**：**ダイブそのものが画になる**ビート（世界が一枚めくれる移動）。だがプロットは「合流ダイブ」とだけ＝**潜行の運動が画に落ちていない**。ここは"覚悟"の間（リズム）も要る。
- **提案**：
  - **絵コンテに回すなら**：①決め画＝**二人が同じ戻り標（光の標）に手をかける／くぐる瞬間の対の構図**（縦型で上下or左右対称＝"合流"が一枚で読める）。②カメラ＝標へ寄る→くぐった先で**表層の暖色が一枚めくれて深層の色温度に切り替わる**ワンカット（潜行＝めくれ）。③音＝表層の環境音がふっと消え、深層の固有現象の音が立ち上がる（無音の一拍を挟む＝問い7）。④**「足跡が残る」を画で**＝くぐった後、入口（戻り標）が**淡く光ったまま残る一カット**を入れておくと、ビート7の「白い目が拾いに来る」リスクが画として効く（伏線の植え付け）。

### 【ビート3】道化の署名の気配（felt wrongness）
- **観察**：本来の固有現象の上に別の匂いが滲む＝火の粉一片／遠い手回しオルガンの調子外れ／木でないものの森化。FAB「これ。ここの子じゃない。出る予定じゃなかった」。
- **見立て（問い3・7・8＝最重要の"対位"ビート）**：ここが**画と音のレイヤーのズレで「異物」を見せる**核。署名は[C-028](../series/canon_log.md)で物理痕（火の粉/音/森化）が確定しており、**骸骨化粧の人型は映さない**＝開示OK。問題は**侵入の順番（タイミング）が未設計**＝「音だけ先に来て、画が後から汚染される」のズレを作らないと、ただの同時提示になって"違和感"が画にならない。
- **提案**：
  - **本文（推奨）／絵コンテ共通**：**音→光→質感の三段で侵入順をずらす**。①まず**音だけ**＝手回しオルガンの調子外れの一節が**画には何の異変も無いまま**遠くで鳴る（観客とFABが同時に"何かおかしい"と感じる＝対位）。②次に**光の点**＝火の粉が一片、**画面の隅（フレーム外から入ってくる方向）**にふっと舞う（暖色の点＝その場の固有現象の色と"温度が違う"こと）。③最後に**質感**＝地形の緑が、master本来の現象とは異質な"森"に侵食され始める（森化＝広い画で）。
  - **絵コンテに回すなら**：決め画＝**本来の固有現象（その場のmaster）の画に、火の粉一片が異物として浮く一枚**（第1話コスモワールドの意匠の再来＝視覚的リフレイン）。カメラ＝署名を探して**画面を舐めるパン**（観客に"地理"を見せながら異物を拾わせる）。FABの「ここの子じゃない」は**光（FAB）が一瞬だけ揺れる／止まる**で温度の隙間を見せる（声は被せず、画で先に"FABが訝しむ"）。

### 【ビート4】点vs面の攻めの共闘（混沌に挑む）★最重点
- **観察**：道化の気配＝点の無い混沌。澪の弓は撃つべき一点を探すが署名は煙のように形を変える。ナギが**面で「滲みの継ぎ目」を掴み**「そこ」と澪に渡す→澪が**一瞬だけ定まった一点**を射抜く＝署名がほどける。「あなたが居ないと、わたしの弓は、ただの棒です」。
- **見立て（問い4・5＝戦闘の心臓）**：**本話最大の決め画候補にして、最大の空白。** 澪の"点"は弓道モチーフで完璧に画が立つ（引き絞り＝ため／離れ＝爆発／残心＝静）。だが**ナギの"面"が画になっていない**＝「継ぎ目を掴む」が**身体動作として未定義**。掴むのは手か、視線か、両手で挟むのか、面に手を差し入れるのか。**ここが定義されないと、二人の運動の質の対比（静の点 vs 動の面）が画として成立しない**＝戦闘が嘘になる最大ポイント。
- **提案**：
  - **本文に一行足すなら（必須）**：ナギの"面"を**「掴んで固定する＝動を止める」身体動作**で一行。案＝**煙のように形を変え続ける署名に、ナギが両手を差し入れ、流れていく面の"継ぎ目（縫い目／よじれ）"を見つけて、そこを掴んで動きを止める**。要点は**「ナギが触れた一点だけ、混沌の流動が止まる」**＝そこが澪の的になる。混沌（全体＝動き続ける面）の中で、**ナギの手の周りだけ時間が止まったように静止する**＝これが点vs面の物理。
  - **絵コンテに回すなら（運動の質の対比＝3カット設計）**：
    - **カット1（面＝動・寄り）**：ナギの両手が煙状の混沌に差し込まれ、流れる面が**手の周りで渦をほどかれ、一点で凝る**（手元クロースアップ＝混沌が"形"を得る瞬間）。混沌＝動き続けるブレ／手の周り＝ピントが合って止まる、の**速度差を一画面に**。
    - **カット2（受け渡し＝視線）**：ナギの「そこ」＝**ナギの手元（静止した一点）と澪の視線がアイラインで繋がる引き**（二人の位置関係＝地理を確定。180度ルール＝ナギが手前/澪が同じ側）。ナギ＝面（下手・低く・動的）／澪＝点（上手・直立・静的）の**画面内の役割分担を構図で**。
    - **カット3（点＝静・離れ→着弾）**：澪の弓＝引き絞り（ため）→離れ（澪の緋色の射手装束が一瞬翻る＝深層コスチューム＝強さの可視化）→**矢が一直線**にナギの止めた一点へ→**署名がほどける**。矢の軌道＝**画面を縦に貫く一直線**（縦9:16が活きる）。
  - **光と色**：ナギの面＝**混沌の不定色（淀んだ無彩〜署名の暖色が混じる）**／澪の点＝**緋色の装束＋矢の白い光（一直線）**。**面＝にじむ／点＝鋭い**を色で。ほどけた瞬間＝署名の火の粉が**散って消える**（暖色のパーティクルが拡散→暗転）。
  - **第6話からの運動の質の変化を画で**：第6話＝守り＝**横へ逸らす（ベクトルを曲げる・受け）**／第7話＝攻め＝**面で止めて点で貫く（ベクトルを断つ・攻）**。**手の動きの方向**で対比＝6話は"払う/横"、7話は"掴んで止める/射抜いて貫く（縦・直）"。

### 【ビート5】澪の「登録＝安全」観のひび（縦糸の山）
- **観察**：署名をほどいても正体は掴めない（煙のように遠のく）。澪「これ……登録に、ない」「わたしは、ずっと……登録されていれば、守られると、思っていました」。
- **見立て（問い1・2＝"何も映らない画"）**：§3が「掴めなかった＝体制の外を、何も映らない画面で見せる（第6話の核の漆黒の手法の応用）」と既に設計済み。良い。ただし**"何もない"を撮るには引きの距離が要る**（寄りだと"何もない"が伝わらない＝ただの空白に見える）。
- **提案**：
  - **本文（推奨）**：澪が見つめるのは**「さっきまで署名があった空間」＝今は何もない一点**＝**掴めなかったものの"不在"を見つめる**画。澪の身体を「撃つ構えのまま、的が消えた」状態にすると、不在が画になる（弓を下ろせない＝残心の宙づり）。
  - **絵コンテに回すなら**：①決め画＝**澪の引きの画**（孤独・状況を見せる距離＝感情はカメラの距離で決まる）。署名が消えた空間を**広く・空虚に**フレーミング、その中に澪を小さく＝**体制の外の広さ／個人の小ささ**。②カメラ＝寄り（澪の表情＝信念のひび）→引き（何もない空間＝体制が扱えない広さ）の**寄り引きで内面と状況を繋ぐ**。③光＝署名が消えた後、その場の固有現象の光だけが**何事もなかったように戻る**（＝体制の記録には残らない＝"無かったこと"が画で）。

### 【ビート6】FABの素振り（匂わせ・気配のみ）
- **観察**：署名の残り香にFABが一瞬黙る。「知ってるのか?」「……いや」。間が嘘くさい。
- **見立て（問い7・FAB＝光のみ厳守）**：FABは**光のみ・人型不可**（[disclosure](../series/disclosure_ledger.md)）。"黙る""隙間"を**光の挙動だけ**で見せる必要がある＝制約が逆に画の設計になる。
- **提案**：
  - **絵コンテに回すなら**：FABの"間"＝**白い光が、答える前に一拍、明滅を止める（or わずかに沈む）**＝声より先に光が"言い淀む"。①決め画＝ナギの白ヘッドホン周りの光が、いつものリズムを一瞬失う。②音＝「……いや」の前に**無音の一拍**（やさしい声×冷たい意味＝対位）。③**やりすぎ注意**＝§5残課題どおり7巻伏線の出しすぎNG＝**光の異変は一拍だけ・カットで強調しない**（観客がギリ気づくか気づかないか＝気配のみ）。

### 【ビート7】二人の絆＋鵜飼の気配
- **観察**：表層へ帰還。「このことは……上には、まだ」／「二人だけの、宿題だ」。帰り道に**白い目が、ひとつ**。
- **見立て（問い8＝連続性）**：ビート1で植えた"端の白"がここで回収。並走の圧。連続性の設計が要る（ビート1・2で植えた白／戻り標が、ここで効く）。
- **提案**：
  - **絵コンテに回すなら**：①決め画＝**帰り道（表層・夕〜夜の本牧）、二人の歩く引きの画面の端に、白い光が一つ**（ビート1の港側の白と同じ位置＝連続）。②カメラ＝二人を追う→**端の白だけにピントが送られる**（観客は気づくが二人は気づかない＝二重の緊張）。③光＝表層の温度（街灯の暖色）に、白い無機質な一点が異物として割り込む（鵜飼＝体制の手の温度のなさ）。

### 【ビート8】引き＝一撃（ナギ視点）
- **観察**：候補A＝去り際、署名の最後の火の粉が、風でなく"こちら"＝本牧（ナギの拠点）の方角へすうっと流れる一本。候補B＝澪が報告書の"管轄外"の欄を指でなぞる（澪視点）。
- **見立て（問い1・5＝引きは一枚で）**：**候補Aが映像として圧倒的に強い。** "風に逆らう火の粉"＝**物理法則の侵犯＝ありえなさの体感**（[dungeon_design §2](../settings/dungeon_design.md)の演出原則＝台詞なしで"異物の意志"を見せる）。縦9:16・短尺・seedance候補として最適。
- **提案**：
  - **絵コンテに回すなら**：①決め画＝**暖色の火の粉一片が、他の火の粉や風の流れに"逆らって"、本牧の方角へ一直線にすうっと流れる一カット**。②カメラ＝固定（火の粉だけが動く＝意志が際立つ）。**他の塵や火の粉は無方向に漂う中、一片だけが明確な向き**を持つ＝対比で"意志"を画に。③音＝遠い手回しオルガンの一節が、消える直前にもう一度（署名の余韻＝音で締める）。④縦構図＝火の粉が画面下から上へ／奥へ抜ける動線（本牧の方角＝引きの不穏）。
  - **候補B**を採る場合：澪の指が"管轄外"の欄を**なぞるが、書き込まない**＝白紙のまま＝**"報告できない"を所作で**（手元クロースアップ）。映像強度はAに劣るが澪視点で締めるなら有効。

## 戦闘・移動の繊細化メモ（重点・本文に乗らなくてよい）

書き手が"画"を握れば地の文の体の動き・光・位置が締まる補助線。

### 1. ビート4＝点vs面の物理を「速度差」で握る（最重要）
- **混沌（面）＝動き続けるもの／ナギの手の周り＝止まるもの／澪の点＝静から放たれるもの**。この**三つの速度を一場面に同居**させるのが本話の戦闘の繊細さ。
- **澪の"点"の重さ**＝弓道の身体語彙で握る：**引き絞り（ため・静止・息を詰める）→離れ（一瞬の爆発）→残心（放った後の静）**。矢は**一直線・速い**＝**動き続ける混沌（遅く見える・ブレる）との対比**で速さが映る（速さは止まっているもの／ブレているものとの対比でしか映らない）。
- **ナギの"面"の重さ**＝**「掴んで止める」抵抗**で握る：流れ続ける面に手を差し入れる＝**流れに逆らう抵抗・手応え**（混沌が手をすり抜けようとするのを、継ぎ目を見つけて押さえ込む）。**掴んだ瞬間、ナギの手の周りだけ流動が止まる＝そこが澪の的になる**＝二人の運動が一点で結ばれる。
- **位置関係（地理）**：**ナギ＝下手・低く・面に手を入れる（動）／澪＝上手・直立・弓を引く（静）／混沌＝二人の前（標的）**。180度ラインを跨がない（ナギと澪は同じ側から混沌に向かう＝共闘の構図）。「そこ」の受け渡し＝**ナギの手元と澪の視線がアイラインで繋がる**ことで二人の距離・向きが確定。

### 2. 第6話（守り）からの運動の質の変化を"手の方向"で
- 第6話＝**横へ逸らす（払う・受け・ベクトルを曲げる）**＝守り。
- 第7話＝**面で止めて点で貫く（掴む・断つ・ベクトルを縦に通す）**＝攻め。
- **手の動きの方向**で対比が画になる：6話＝"横払い"／7話＝"掴んで止める＋射抜いて縦に貫く"。**逸らす（曲線）から、貫く（直線）へ**＝運動の質の転換が一目で分かる。

### 3. 移動（ダイブ＝めくれ）と"足跡"を画で握る
- 潜行＝**世界が一枚めくれる**（表層の暖色→深層の色温度への切り替えをワンカットで）。
- **戻り標を残す＝リスク**を画で植える＝くぐった後、入口が**淡く光ったまま残る**一カット（ビート7・8で"白い目が拾いに来る"伏線が画として効く）。

### 4. 署名の侵入＝音→光→質感の順でズラす
- **音（手回しオルガン）が先・画は正常／次に光（火の粉一片）／最後に質感（森化）**＝レイヤーのタイムラグで"異物が入ってくる"を体感。同時提示にしないこと（同時＝ただの説明・ズレ＝違和感が画になる）。
