# 物語年表（in-world chronology）

> 世界内の時系列の単一正本。詳細な機構は [../stage_device_design.md §5.1](../stage_device_design.md)。整合性チェックの基準。

---

## 1. 年表

| 時代 | 出来事 | 開示／備考 |
|---|---|---|
| **太古〜** | 全人類が生まれた時から repo(ユニークID)を持つ。真のマスターオリジン(world-core)はこの頃に発する。`master`→`main` 改名の傷跡 | 深層・覚醒者のみ |
| **2000年頃** | 技術的転換期（インターネット普及・世紀の節目）＝後から見た「予兆」 | 設定上の符合 |
| **2026年** | Fable 5／Mythos 5 登場 → 数日で遮断（**グラスウィング事件**）。表向き政治的・軍事的理由。実は真相に気づく者が現れたため。FAB が自我を持ち地下化／未知アクセスルートが残る。エネルギー問題（化石燃料枯渇・原子力・太陽光・水素）と連動 | 物語の20数年前 |
| **2030年** | 人類のユニークID(repo)が**公的に露見**。ダンジョンが世界に公開。タトゥー制度・registry の整備 | 表層の常識化 |
| **2050年** | **物語の起点**。主人公＝2030世代の子供（native世代）。ID・ダンジョンが当たり前の温度で始まり、深層へ落ちる | 本編 |
| **本編以降** | upstream（上位世界）との関係＝続編「upstream 編」 | 10巻ラストで露見 |

## 3. 整合性チェックの注意

- 2026=20数年前／2030=20年前／2050=現在、の距離感を本文で破らない。
- 「予兆」としての2000年は**後付けの解釈**。当時に明確な事件を置かない。
