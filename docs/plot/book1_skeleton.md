## 1. 章ブロック構成

### 第2ブロック「追われる者／最初の仲間」＝B1-05〜10（未着手・目安）
- **監視機構の末端＝追跡者**が**戻り標を辿って**ナギの拠点へ迫る（[ep3 §5＝入口だけは残る](ep3_plot.md) を回収）。
- 澪との関係深化＝共有する宿題「"管轄外"のイレギュラー」を二人で追う。澪の「報告保留」が体制に**露見し始める**圧。
- 削れの天秤を**揺らし続ける**（新要素を増やしすぎず、既出＝拠点・澪・削れ・道化の気配を深掘り＝ARC-01 課題E）。

### 第3ブロック「初めての決着」＝B1-11〜16（未着手・目安）
- **追跡者との決着＝巻1クライマックス**。**撃破でなく「出し抜く／なかったことにさせない」**（致死の抑制・著者要件）。点vs面の連携が完成形で噛む。
- 澪が**正式に最初の仲間**になる決め手の出来事（職務を踏み越える選択）。
- 削れに頼るか自力で通すかの**天秤が一度大きく傾く**山場（ただし最終決着は巻2以降へ未決で残す）。

### 巻末ブロック「次への扉」＝B1-17〜19（未着手・目安）
- 末端の背後に**"上"（4学派・統制機構）の影**が一瞬差す（正体は伏せる＝気配のみ）。
- **道化の楽団の気配の再来**（B1は気配のみ・[C-028](../series/canon_log.md)）。
- 削れの天秤を**未決のまま巻2へ持ち越す**引き。

## 1.5 巻1の当面の敵＝追跡者「鵜飼」（暫定確定 v0.1）

> [series_arc §1](series_arc_draft.md)／[factions §4](../settings/factions.md) の「監視機構の末端」を**1体に具体化**した暫定確定（2026-06-15・[canon_log C-033](../series/canon_log.md)）。**人物像・能力・名は仮**（語感・展開で差し替え前提）。第2〜第3ブロックの執筆ブロッカー解消が目的。

### 0. 一目で
| 項目 | 暫定値 |
|---|---|
| **立場** | 中央評議会／統制機構の**末端監察官**（無origin派の下流実働）。**澪と同じ「登録・監視」系の下の人間**＝体制の善人ではなく冷徹な実務者 |
| **通称（仮）** | 「足取りの鵜飼（うかい）」。鵜（観測ノード）を放って獲物の動線を辿る意 |
| **型** | **足跡読み＝観測者/射手型**（[battle §5 観測戦](../settings/battle_system.md)）。本人の戦闘力は低く、**観測優越がすべて** |
| **能力①トレース** | 帰還アンカー（**戻り標**）の残響を辿り、覚醒者の動線・拠点を割り出す（[battle §1-3](../settings/battle_system.md)） |
| **能力②抹消申請** | 見つけた対象に「**なかったことにする**」タグを打つ＝強制 sync の烙印。通れば**拠点と深層活動の記録が revert（現実巻き戻し）**＝[battle §4-2 crash（非致死）](../settings/battle_system.md)・[factions §4 抹消・隠蔽](../settings/factions.md) |
| **キャラの芯** | 官僚的で冷たい。**「君を消すのは悪意でなく、必要だ」**＝議長センの末端版（"上"の体温のなさを先に体感させる） |
| **弱点** | 動線が読めなければ無力。**デッドゾーン化・盲目化に弱い**。抹消申請の裏取りに**澪ら面側の証言が要る**＝澪の「報告保留」が直接効く |
| **致死性** | **非致死**（記憶・築いた現実の喪失止まり）。著者要件の致死抑制と整合。撃破対象でない |

### 1. 「出し抜く／なかったことにさせない」決着の機構
撃破でなく出し抜く決着が、この設計だと**機構的に必然**になる（[ARC-01 課題C](book1_structure.md) を解消）。

1. **ナギ＝点**（歩く exploit・デッドゾーンに隠れる）と**澪＝面**（監視網側）の点vs面連携が完成形で噛む（[ep2 で立てた軸](ep2_plot.md)）。
2. ナギは**囮の戻り標**で鵜飼のコンステレーションを誘い出し、本物の拠点を**デッドゾーン化で盲目化**。
3. 澪が**職務を踏み越え**、鵜飼の抹消申請の裏取り（面の証言）を止める／囮へ誘導＝**澪の正式仲間化の決め手**（第3ブロック）。
4. 結果＝鵜飼の抹消申請が**空振り**。拠点と「澪が見逃し続けた記録」が**revert されずに残る**＝**なかったことにさせない＝初の明確な勝利**。**鵜飼は退く（撃破でない）が、背後に "上"（4学派・統制機構）の影が差す**＝巻末ブロックへ接続。

### 2. ブロック別の配置（目安）
- **第2ブロック（B1-05〜10）**：鵜飼が戻り標を辿って拠点へ迫る圧。**初遭遇は気配・痕跡から**（顔出しは段階的に）。澪の報告保留が体制に露見し始める伏線と連動。
- **第3ブロック（B1-11〜16）**：鵜飼との**決着＝巻1クライマックス**。上記の出し抜く機構が噛む。
- **巻末ブロック（B1-17〜19）**：鵜飼の背後に "上" の影（正体は伏せる＝気配のみ）。
