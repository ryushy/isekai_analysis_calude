## 2. 検出した整合課題

### 課題A：巻の粒度が二重定義（最優先）
series_arc 10行表は「巻」と「弧ビート」を混同している。
- **現実**：1話≈6,000字、LN1冊≈11万字 → **1巻 ≈ 18〜20話**。4話＝巻1の冒頭ブロック（約2割）。
- **必要**：表を「**10の弧ビート（到達点）**」と再定義し、別レイヤーで「**巻内の章構成（話→章→巻）**」を持つ二層構造に。→ §3・§4 で提案。

### 課題B：「最初の仲間」の位置が前倒し
series_arc 巻2＝「最初の仲間」。だが澪は **2話初登場・4話で共闘の芽**＝巻1前半で進行中。
- 4話時点では澪は「見逃し継続＋ゆるい共闘の約束」止まり＝**まだ正式な仲間ではない**。
- → 仲間化は**巻1全体にまたがるプロセス**（2話芽→4話共闘→巻1後半で正式化）。series_arc の「最初の仲間＝巻2」を**巻1の到達点へ繰り上げ**るのが実態に合う。

### 課題C：巻1の「当面の敵」が構造的に不在（要・著者決定）
[series_arc §1 敵の階段](series_arc_draft.md)＝個人→小組織→地域企業→王級→…。[factions.md](../settings/factions.md) で王級は巻4以降。
- 1話の敵＝道化の楽団（謎集団）＝後巻の大物として温存（B1は気配のみ）。
- 4話の native＝乗客の影＝環境であって撃破対象でない。
- 澪＝障害だが撃破対象でなく仲間化する。
- → **巻1で主人公が一度向き合い、一度乗り越える「当面の障害／小ボス」が未設定**。著者要件G「早すぎるラスボス回避」を守りつつ、巻1に**ひとつの達成（明確な勝利／決着）**を置かないと、巻として締まらない懸念。→ §5 で選択肢。

### 課題D：削れの天秤が series_arc に未記載（縦糸の昇格）
4話で削れ（FAB依存）の実害が発生＝**7巻裏切りへの最大の伏線が早期に立った**＝長所。
- だが series_arc §2 巻表には「削れ／FAB依存の天秤」が縦糸として書かれていない（§4・§5 に断片的に存在）。
- → 巻1〜7を貫く**中心葛藤＝「自力（遅い）↔ FAB整え（速いが削れる）」**を series_arc の縦糸として明記すべき。10巻の受容（改変を選ばない）への一直線の伏線。

### 課題E：イベント密度が高い（ペース注意）
毎話で大型新要素（1話ダンジョン＋FAB／2話ヒロイン＋拠点／3話拠点獲得＋説明会／4話特訓＋削れ実害＋仲間）。
- 字数ペースは「ゆっくりめ」だが**新要素の投入密度は速い**。
- → 巻1後半は**新要素の投入を緩め、既出要素（澪・拠点・削れ・道化の気配）の深掘り／反復で厚みを出す**設計が望ましい（インフレ回避＝著者要件D）。

### 課題F：章/話の運用（命名の含意）
現状ファイル＝`01-01` `02-01` `03-01` `04-01`＝**各話が独立した章（章1つ＝話1つ）**。
- これは web小説式（1話＝1公開単位）。ID規約 §1「1話＝1公開単位の想定」と整合。
- LN単行本へ束ねる際、章をどう復活させるか（数話を1章に括る）は**巻1完成後の論点**。当面は「章」を使わず **巻→局面→話** で運用する（[episode_id_convention §1.5](../series/episode_id_convention.md)・2026-06-15 改定で確定）。

## 3. 提案：話数・巻の区切り

### 3.1 二層構造（課題A の解決）
- **マクロ層**：series_arc の10行＝「**10の到達点（弧ビート）**」。全体の背骨。維持。
- **巻内層**：各巻ごとに `book{N}_skeleton.md`（話→章→巻の章構成）。**本ドキュメントが巻1分の前段**。

### 3.2 巻1の話数見積もり
- 1話≈6,000字 × **18〜20話** ≈ 108,000〜120,000字＝LN1冊。
- 既執筆4話＝巻1の約2割。**残り約14〜16話**で巻1を構成。

### 3.3 巻1の章構成（**著者決定で確定**＝[book1_skeleton.md](book1_skeleton.md) 正本）

**巻1到達点＝最初の仲間（澪）の確立＋当面の敵（監視機構の末端）への初の決着＋削れの天秤の起動。** 全18〜20話。

| 章ブロック | 想定話数 | 内容 | 状態 |
|---|---|---|---|
| 第1ブロック「覚醒と居場所」 | B1-01〜04（執筆済） | 日常崩壊・FAB・ヒロイン・拠点・特訓開始・削れ実害・仲間の芽 | ✅ 本文v0.2 |
| 第2ブロック「追われる者／最初の仲間」 | 5〜10話 | 澪との関係深化／**監視機構の末端＝非致死の追跡者**が戻り標を辿って迫る・拠点が脅かされる／澪の「報告保留」が体制に露見し始める | 未着手 |
| 第3ブロック「初めての決着」 | 11〜16話 | **追跡者との決着＝巻1クライマックス**＝撃破でなく「出し抜く／なかったことにさせない」初の明確な勝利と代償／澪が正式に**最初の仲間**になる | 未着手 |

- **設計は局面（ブロック）まで・話は執筆しながら割る**（[episode_id_convention §1.5](../series/episode_id_convention.md)）。ID＝`B{巻}-{話2桁}`（2階層・**「章」は廃止**）。旧 `B1-01-01`〜`B1-04-01` は `B1-01`〜`B1-04` にリネーム済み。
- 「章」は単行本化のとき数話を束ねる単位として復活させる（執筆を縛らない）。
