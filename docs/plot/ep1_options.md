## 2. 組み合わせ案（4案）

### 案1「放課後の廃モール」 ＝ A1 + B1 + C1 + D1 + E2 + F1
- **ログライン**：放課後、同級生に誘われ近所の廃モール（登録ダンジョン浅層）へ。規格外の獣に追い詰められ初覚醒。止まった噴水を「あふれなかったこと」にして切り抜けるが、その瞬間を"剣を持つ少年"に見られている。
- **ビート**：①学校の日常（外来者の目・あざを隠す）→②入口で潜行、歪んだ廃モール（吹き抜け・無人の店・止まった噴水）→③獣の急襲・《アンブレラ》で受けるが容量が焼ける→④加速＝世界が静止→⑤FAB「やあ」初接触→⑥自作技で噴水を止め活路→⑦剣の少年が現れ「君がやったの?」、FAB「あの子は登録もの」→⑧「いい子だから報告する」で引き。
- **映像化◎**：廃モールの吹き抜け＝縦9:16映え。止まった水・凍る埃。日常(学校)→非日常(歪んだモール)の落差。
- **設定影響**：剣の少年＝同級生 or 別人かを早期に決める必要。廃モール＝第1ダンジョンとして登録。喪失は薄い気配のみ。
- **長所/短所**：導線が分かりやすく映像強い／喪失の情緒は後回し（薄め）。

### 案2「川辺の静止した雨」（旧試作の継承・実在地強化） ＝ A3 + B4 + C1 + D2 + E1 + F2
- **ログライン**：採取バイトで入った川辺ダンジョン（橋の下）。記録の雨が降る歪んだ河川敷で獣に襲われ、あざが熱を持つ。雨が静止し、《サンブレイク》で頭上だけ「降らなかったこと」にして勝つが、FABは「負けたら、なかったことになる」と告げる。
- **ビート**：①歪んだ河川敷・橋脚・本物でない雨→②獣の急襲・《アンブレラ》→③容量超過寸前→④あざが熱→⑤雨が静止（加速）＋FAB滲み出る→⑥《サンブレイク》deploy＝半径2m乾く→⑦crashの警告「なかったことになる」→⑧自分の世界の違和感で引き。
- **映像化○**：旧試作「静止した雨」を実在地（橋の下）で再構築＝場所感UP。水の静止が看板ビジュアル。
- **設定影響**：既存3試作と地続き（流用効率高）。剣の少年は第2話送り。あざの反応を最序盤で見せる。
- **長所/短所**：既存資産・看板絵を活かせる／川辺は地味になりがち（モール/駅より画の派手さ弱め）。

### 案3「駅の地下、二重生活」 ＝ A2 + B2 + C2 + D3 + E2 + F3
- **ログライン**：留学先の朝（日本食・あざを隠す）から始まる日常。放課後、同級生が野良の入口に迷い込み、庇って潜行。歪んだ地下駅（止まったエスカレータ・乱れた案内）で覚醒。「ヘッドホンの声」だと思っていたFABが正体を現す。日常に戻るが、昨日あった温もりの記憶が一つ、消えている。
- **ビート**：①自宅の朝→②学校→③同級生が入口に巻き込まれる→④庇って潜行・歪んだ地下駅→⑤危機でFAB（実はずっと耳元にいた）→⑥エスカレータを「動かなかったこと」に→⑦帰還・日常→⑧失われた記憶（削れの予兆）で引き。
- **映像化◎**：地下駅＝迷宮・縦構図・止まった群衆。日常↔深層の往復が一話に収まる。白ヘッドホン＝FABの伏線が画で効く。
- **設定影響**：FAB＝既知接触型に倒す（fab_persona の接触作法に影響）。削れ＝1話で予兆を出す（早い）。同級生＝相棒候補。
- **長所/短所**：主人公の魅力・二重生活・FAB伏線が濃い／情報量多め（1話が忙しい）。

### 案4「団地の亡霊」（情緒重視） ＝ A1 + B3 + C1 + D1 + E3 + F4
- **ログライン**：誘われて入った古い団地ダンジョン。生活の亡霊（洗濯物・無人の部屋・誰かの夕餉）がにじむ歪んだ住宅街で獣に襲われ、あざが暴走しかける。FABが制御を手伝い辛くも切り抜けるが、その"親切"の奥に、人の温度のない何かを感じる。
- **ビート**：①学校→②歪んだ団地（生活の残響）→③獣・《アンブレラ》→④あざ暴走→⑤FABが制御補助（初接触＋不穏）→⑥辛勝→⑦FABの目的の影で引き。
- **映像化○**：団地の生活感×非日常＝独特の情緒。だが画の派手さは中。
- **設定影響**：あざ暴走＝心意級の縁を早く見せる（重い／開示注意）。FABの不穏を1話で出す（7巻裏切りの初伏線を前倒し）。
- **長所/短所**：情緒・テーマ性が高い／暗め・地味・開示が前のめりになりやすい。

## 4. 設定全体への影響（共通の論点）

- **剣の少年をいつ出すか**：1話ラスト（案1）か第2話（他案）。早出しは「監視＝体制」を強く示すが情報増。
- **FABの接触型**：初対面（D1）／あざ呼応（D2）／既知（D3）。**fab_persona の接触作法**に波及。D3 は白ヘッドホン伏線が強い。
- **削れの予兆**：1話で出す（案3）と早い。喪失の核（protagonist_core）は伏せたまま"気配"で。
- **第1ダンジョンの実在地**：選んだ案がそのまま「最初の舞台」として canon 化（実在地タイプの第1号）。
- **既存3試作の扱い**：案2 は流用、他案は「ひと晩寝かせる（特訓）」を後日譚に回すなど再配置。

## 5. 推奨（参考）

- **映像インパクト＋分かりやすさ重視**＝案1 or 案3。
- **既存資産活用＋世界観の看板（静止した雨）**＝案2。
- **テーマ・情緒先行**＝案4。
- **折衷の例**：案3 の「自宅の朝→学校→潜行→二重生活」の枠に、案1 の廃モール（画の派手さ）と案2 の《サンブレイク》（看板ビジュアル）を載せ、剣の少年は第2話へ——という混成も可能。

> **決め方**：まず ①どの実在地（B）②FABの出方（D）③締めの引き（F）を選べば、骨子が決まる。
