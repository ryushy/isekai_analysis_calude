## 0. ログライン

みなとみらいでの"無登録の潜り"は、見られていた。剣を持つ同年代の少年が、監視の糸として近づいてくる――やさしい顔で、「報告する」と言いながら。追われたナギは、駅のない街・本牧で、現実には無いはずの駅へ迷い込む。誰の目も届かない、ひとりだけの隠れ場所へ。

## 1. この話の目的

- **監視テーマの起動**：無登録の力は"見られる"。第1話の「戻り標の脆さ」を回収。
- **剣の少年（登録監視・体制側の善人）登場**：敵でない障害＝重厚さ（prototype「剣のひと」を本採用）。
- **魔法 vs 剣＝能力層の提示**（軽く・格差を見せる）。
- **拠点の獲得＝本牧の幻の地下鉄駅**（監視外の隠れ家・今後の訓練場）。
- **FAB 依存の入口**：訓練を"手伝う"申し出＝削れの誘惑（第1話の喪失と地続き）。

## 2. ビート（出来事＋手触りメモ）

| # | 場面（出来事） | 五感／内面／会話／戦闘の手触り（仕込み） |
|---|---|---|
| 1 | **本牧の朝・余波** | 五感＝海の匂い・バス待ちの空気。内面＝"声"が誰かだと知ってしまった落ち着かなさ／昨日の小皿は今も思い出せない（削れの不安）。会話＝FABと心のなかで一往復（信用と不気味さ） |
| 2 | **学校（山手駅）・視線** | 五感＝鐘の塔の音・廊下の気配。内面＝見られている感覚。会話＝FABの警告「昨日のあれ、登録されてない潜りだった。気づかれてる」（監視を現象として・git非表示） |
| 3 | **剣の少年 登場** | 五感＝右腕の青い線がぼうっと光る・雨上がりの匂い。会話＝穏やかに「君だよね、あそこに入ったの」。内面＝同年代に初めて"同じ側"を感じる戸惑い |
| 4 | **対峙（圧・デモ）** | 戦闘の手触り＝少年が腕の線を剣の形に具現（金属でなく光）。ナギがとっさに書いた小さな守り（《アンブレラ》）を、剣が触れて"無かったこと"にする＝巻き戻し。内面＝「剣がこわいんじゃない。剣の形をした、同じ種類の力がこわい」。会話＝「いい子だから――君のことを、ちゃんと上に報告する」 |
| 5 | **逃走・惹かれ** | 身体感覚＝息・坂・バスを降りて走る。内面＝逃げ場を探す焦り。あざが疼く＝駅のない本牧の路地へ、導かれるように（歩く exploit の"惹かれ"） |
| 6 | **幻の地下鉄駅 発見** | 五感＝あるはずのない改札の気配・埃と湿った冷気・止まった時間。内面＝「駅なんて、ないはずなのに」。深層へ＝静かな、誰も知らない空間（廃／未成の駅） |
| 7 | **拠点・FAB** | 会話＝FAB「ここは、誰の目も届かない。練習には、ちょうどいい」。内面＝初めて手に入れた"自分の場所"の安堵。手触り＝薄暗い構内に青い線がともる。**不穏**＝FABの「手伝おうか、もっと速く強くなれる」（削れの誘惑＝第1話の"消えた味"と繋がる影） |
| 8 | **引き** | 内面＝剣の少年は敵か、監視の糸か。拠点は得た。だが「報告」はもう上がっている。会話＝FABの最後の一言で、薄い寒気。次への問い＝"上"とは誰か（伏せたまま） |

## 3. 映像化の見どころ

- 山手駅の煉瓦・鐘の塔（表層）↔ 本牧の路地↔ 幻の地下鉄駅（深層）の落差。
- **剣＝青い光の具現**が、ナギの守りを"なかったこと"にする一瞬（魔法 vs 剣の看板カット）。
- **駅のない街に、ありえない改札**（深層性の体感＝第1話「地下なのに雨」と同系）。
- 廃／未成の駅に青い線がともる隠れ家の絵（今後の訓練拠点のアイコン）。
