## 0. ログライン

みなとみらいでの"無登録の潜り"は、見られていた。遠くから帰り道を射抜く少女が、監視の糸として近づいてくる――引き絞った弓を、ナギに向けたまま、ひと呼吸、引き金を引けずに。追われたナギは、駅のない街・本牧で、現実には無いはずの駅へ迷い込む。誰の目も届かない、ひとりだけの隠れ場所へ。

## 1. この話の目的

- **監視テーマの起動**：無登録の力は"遠くから見られている"。第1話の「**戻り標の脆さ＝帰還地点が割れる**」を、**人格を持った狙撃の脅威**として回収。
- **ヒロイン（登録監視・体制側の善人・見逃す葛藤型）登場**：敵でない障害＝重厚さ（剣の少年案を置換。[heroine_core.md](../settings/heroine_core.md)）。
- **魔法 vs 弓＝能力層の提示**（軽く・遠隔ピンポイント編集を見せる）。
- **拠点の獲得＝本牧の幻の地下鉄駅**（監視外の隠れ家・今後の訓練場。**狙撃から逃れる＝監視外へ**が動線として自然）。
- **FAB 依存の入口**：訓練を"手伝う"申し出＝削れの誘惑（第1話の喪失と地続き）。**人の温度（見逃す彼女）↔ 温度のない FAB** の対比軸を起動。

## 2. ビート（出来事＋手触りメモ）

| # | 場面（出来事） | 五感／内面／会話／戦闘の手触り（仕込み） |
|---|---|---|
| 1 | **本牧の朝・余波** | 五感＝海の匂い・バス待ちの空気。内面＝"声"が誰かだと知ってしまった落ち着かなさ／昨日の小皿は今も思い出せない（削れの不安）。会話＝FABと心のなかで一往復（信用と不気味さ）。さらに**首のうしろの、あの感じ**（第1話ラストの帰還の脆さ＝戻り標が残っている予感） |
| 2 | **学校（山手駅）・視線** | 五感＝鐘の塔の音・廊下の気配。内面＝見られている感覚（誰に、ではなく"遠くから"見られている）。会話＝FABの警告「昨日のあれ、登録されてない潜りだった。**戻り標を読まれてる。帰り道が、もう割れてる**」（監視を現象として・git非表示） |
| 3 | **ヒロイン登場（女子校・弓道）** | 五感＝隣の女子校の弓道場の弦音・凛とした空気。視覚＝制服に弓巻を提げた同年代の少女が、**こちらをまっすぐ見ている**（目を離さない＝監視と狙撃の人格表現）。会話＝静かに「君だよね、昨日あそこに入ったの。登録、してないでしょう」。内面＝同年代に初めて"同じ側"を感じる戸惑い＋射すくめられる感覚 |
| 4 | **対峙（圧・デモ）** | 戦闘の手触り＝少女が腕／弓具の青い線を**弓の形に具現**（金属でなく光の弦）。遠い間合いから一射。ナギがとっさに書いた小さな守り（《アンブレラ》）を、**矢が遠くから撃ち抜いて"無かったこと"にする**＝遠隔ピンポイント編集。内面＝「速いんじゃない。**遠い**。逃げる前に、もう狙われている」。会話＝引き絞ったまま、ひと呼吸――「……いい子だから。**上に報告する**。それだけ」。だが**指が、すぐには引き金を引かない**（見逃す葛藤の最初の一瞬） |
| 5 | **逃走・惹かれ** | 身体感覚＝息・坂・バスを降りて走る・**遮蔽を選んで走る**（遠隔狙撃から逃げる＝物陰へ物陰へ）。内面＝逃げ場を探す焦り。あざが疼く＝駅のない本牧の路地へ、導かれるように（歩く exploit の"惹かれ"） |
| 6 | **幻の地下鉄駅 発見** | 五感＝あるはずのない改札の気配・埃と湿った冷気・止まった時間。内面＝「駅なんて、ないはずなのに」。深層へ＝**ここなら遠くから狙えない・誰も知らない空間**（廃／未成の駅＝監視外） |
| 7 | **拠点・FAB** | 会話＝FAB「ここは、誰の目も届かない。**戻り標も読まれない**。練習には、ちょうどいい」。内面＝初めて手に入れた"自分の場所"の安堵。手触り＝薄暗い構内に青い線がともる。**不穏**＝FABの「手伝おうか、もっと速く強くなれる」（削れの誘惑＝第1話の"消えた味"と繋がる影）。**対比＝彼女は引き金を引かなかった（人の温度）／FABはやさしいのに温度がない** |
| 8 | **引き** | 内面＝あの子は敵か、監視の糸か。なぜ撃たなかったのか。拠点は得た。だが「報告」はもう上がっているかもしれない。会話＝FABの最後の一言で、薄い寒気。次への問い＝"上"とは誰か（伏せたまま） |

## 3. 映像化の見どころ

- 山手駅の煉瓦・鐘の塔（表層）↔ 隣の女子校・弓道場の弦音↔ 本牧の路地↔ 幻の地下鉄駅（深層）の落差。
- **弓＝青い光の弦**を引き絞る一瞬／**遠くから放たれた一矢**が、ナギの守りを"なかったこと"にする（魔法 vs 弓の看板カット）。
- **引き絞ったまま、引き金を引けない指**（見逃す葛藤の絵＝関係性の核）。
- **遮蔽を縫って逃げる**ナギ（遠隔狙撃から逃げる＝近接バトルと違う絵）。
- 駅のない街に、ありえない改札（深層性の体感＝第1話「地下なのに雨」と同系）。
- 廃／未成の駅に青い線がともる隠れ家の絵（今後の訓練拠点のアイコン）。
