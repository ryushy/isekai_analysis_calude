## 0. 方針

- 単行本**10冊相当**の重厚なストーリー。**早すぎるラスボスを回避**し、続編余地を残す。
- 手法（naro 模倣）：**敵の層を階段状に上げる ＋ 謎を一枚ずつ剥がす**。

---

## 1. 敵の階段（早すぎるラスボス回避）

**個人 → 小組織 → 地域企業 → 学派の王 → 監視装置/Glasswing → 人間の黒幕(偽origin運用者) → 「中心という誘惑」**

本当のラスボスは人ではなく **「中心があるという幻想・自らが中心になる誘惑」**。だから最後まで "倒して終わり" にならない。

**巻1の当面の敵＝「監視機構の末端」**（ARC-01 決定 2026-06-15）：最下段「個人〜小組織」に当たるのが、**澪の背後の体制（無origin派・統制機構）の末端実働＝非致死の追跡者**。戻り標を辿ってナギの拠点へ迫る"足跡読み"の追っ手。**撃破でなく「出し抜く／なかったことにさせない」決着**＝巻1クライマックス。"上"（4学派・偽origin）の正体は伏せたまま、その**末端の手**だけを巻1で当てる（[factions.md §4](../settings/factions.md)／[book1_skeleton.md](book1_skeleton.md)）。1話の道化の楽団は後巻の大物として温存。

## 2. 全10巻（git 開示度＝徐々に明かす伏線）

> **【粒度の注記（ARC-01 2026-06-15）】この表の各行は「単行本1冊」ではなく「弧ビート（到達点）」。** 実際の1巻 ≈ **18〜20話**（1話≈6,000字×LN1冊≈11万字）で、1冊は複数の弧ビートを束ねる。**巻内の章構成は別レイヤー**＝[book{N}_skeleton.md](book1_skeleton.md)。例：巻1は本表の「1（日常〜FAB）」＋「2（特訓・最初の仲間）の前半」を統合し、**最初の仲間（澪）の確立＋当面の敵への初の決着**で締める（[book1_structure.md](book1_structure.md)）。

## 3. 謎の開示スケジュール（X 伏せ）

- 偽origin の存在：3巻で確信／6巻で輪郭
- 主人公＝CVE-0 の正体：3巻示唆／6-7巻核心
- **相棒 FAB の真の動機：7巻露見／10巻で最終**
- マスターオリジンの正体：9-10巻
- **upstream＝上位世界（続編の核）：10巻ラストまで完全隠蔽**

---

## 4. 重厚さの源

- **4学派それぞれに理がある**＝モラルの多義性（単純な善悪にしない）
- **deploy＝他者の現実と記憶を書き換える倫理的 horror**（勝つこと自体が罪になりうる）
- **crash の喪失**（記憶喪失＋現実巻き戻し。失った記憶は append-only ゆえ戻らない）
- **個人→社会→制度→宇宙** の段階拡大（codex の goal_expansion を10巻かけて昇る）
- **FAB の異質な動機**（人 help ではなく "バグを報告し切りたい" 等、最後まで読み切れない相棒）

---

## 5. 続編の余地

10巻は **偽origin の崩壊と原契約の回復で大団円**——しかし**到達した瞬間、"upstream"（この世界自体が上位 repo の fork）が露見**。上位世界派が正しかった。解決が、より大きな扉（世界の外／AI の transcendence）を開く。**続編＝"upstream 編"**。
