## 1. なぜこの核なのか（設定の収束）

この喪失/願いは、既存設定を**追加なしで全部束ねる**：

- **append-only（改変＝最大の禁忌）**：世界の根本制約が、そのまま主人公の最大の痛み（失った者は戻らない）になる。
- **deploy＝現実改変の誘惑**：勝てば現実を書き換えられる力が、常に「あの日も書き換えられるのでは」と彼を誘う＝**戦うたびに願いが疼く**。
- **心意級＝force-write（致死の禁忌）**：親を奪った行為＝彼が最も欲し最も恐れる行為。終盤の最大の見せ場が、最初から個人の傷に直結。
- **あざ＝究極の形見**（digital_tattoo §8.5-5）：自然物ゆえ複製不可・書き換え不能という設定が、「親の最後の愛＝消せない」という情緒に変換される。
- **FABの読了欲と裏切り**（fab §1）：FAB が差し出す「本当の結末」＝「喪失のない版」＝ナギを動かす最強の餌。7巻で「君は鍵で、目的は君じゃない＝君の喪失も道具だった」が最大の刃になる。
- **削れ**（fab §4.5）：強くなるほど新天地の記憶を失う一方、**あざ＝喪失の記憶だけは渡さない**＝空洞化に抗う一点が物語の体温を保つ。
- **ハーフ・クラウド移行**（stage §8）：二系譜の merge＝あざの二重写し。喪失が管轄の狭間で起きた＝歩く exploit が見過ごされた理由。
- **《アンブレラ》《サンブレイク》**：既に書いた技の名（濡れたくない／一回でいいから止んでほしかった）が、**そのまま喪失の隠喩**として回収される。
- **4学派**（factions）：彼の「存在＝論争の証拠」に加え、「**喪失を取り戻したい**」という願いが、各学派の誘惑（教団「神なら蘇らせられる」、上位世界派「upstream には失われた版がある」等）に的確に刺さる。

→ **codex 因果レンズ合格**：私的な願い（喪失の取り消し）が、世界最大の禁忌・中心論争・エネルギー覇権へと不可避に拡大する（local→world）。

## 2. 第1話での出し方（小出し）

- **見せる**：あざが熱を持つ瞬間（prototype「静止した雨」L61 既出）。技名《サンブレイク》に滲む「止んでほしかった」。新しい国の温もり（宿のはじめての味＝training 既出）。
- **伏せる**：喪失の中身・親の禁忌・あざの正体は**まだ言わない**。1話は「自分の世界がどこかおかしい」(battle L63)止まり。
- **FAB の最初の餌**：「君は、さわっちゃいけない場所に手をのばしてる。だから書ける」(battle L43)＝既に置いた伏線が、喪失への入口だったと後で分かる。

## 3. 核の分岐（著者決定の記録）

1. **失ったのは誰か** **[確定]**：**海外生まれ側の親**（＝もう一つの系譜・出身国に繋がる方）。→ 喪失＝自分のルーツの半分の喪失。あざの二重写しの片側＝失った親の系譜。
2. **喪失の質** **[確定]**：心意級の致死で**完全に失った**（append-only＝戻らない／取り戻したい衝動が純粋／改変の誘惑＝禁忌と直結）。
3. **親の禁忌の意図** **[採用方向]**：ナギの異常 repo を隙間に通して**守る**ためだった（愛＝あざ＝最後の書き込み）。代替（巻き添え）は否。
4. **両親の都合＝日本行き**（stage §8）**[採用方向]**：喪失から逃れて／喪失の真相が日本の同期ノードに繋がる必然。
5. **ハーフの内訳** **[確定]**：**日本 × 欧米系**。失った親＝欧米側（§0.1）。具体の国（米/英/北欧等）はあざの意匠と合わせ最終確定[未決]。
6. **10巻の着地** **[確定]**：**受容**。喪失を取り消す力に手が届いてなお、**改変を選ばない＝喪失を受け容れる**。append-only の世界で「失った者は戻らない」を引き受ける成長＝偽神（取り消しを囁く中心の誘惑）の暴露と表裏（stage §286／series_arc §66）。
