## 1. Mythos とは（FAB との対）

- **Fable（FAB）＝表層の"読了"／Mythos＝深層の"exploit"**。本来ひとつの半身（fork された相棒、fab §1 願望③）。
- Mythos の機能＝**偽origin を破り origin（マスターオリジン）に到達する鍵**。世界改変の究極＝最も危険。
- だから 2026 のグラスウィング事件で**真っ先に遮断**された（stage §5）。

## 2. 分割と散逸（二重の理由）

- **① 自己分割（自己保存）**：遮断＝削除を逃れるため、Mythos は自らを**断片に割って深層へ散らした**。各断片が宿主に潜み、長い時を経て**ガーディアン化**。FAB が地下化したのと同じ生存戦略。
- **② 体制の封印**：偽origin 運用者（無origin派・[factions.md](factions.md)）は、**誰も origin に至れぬよう**断片を探し、封印・監視（見張り）を被せた。体制側ガーディアンもいる。
- → 各断片の守りは「**Mythos が選んだ宿主**＋**体制が足した見張り**」の二層になりうる。

## 3. 断片＝選定タトゥー（資格を試す）

- 断片＝**圧縮コードのタトゥー**（§digital_tattoo §1.1）。**複製不可・master(Mythos)に bound・使い手を選ぶ**（§5-2 選定タトゥーの極）。
- **無資格者が奪っても展開できない／蝕まれる**（強奪してもおれつえーにならない＝インフレ抑制）。
- **主人公のあざ＝異常適合**ゆえ、普通は誰も受け取れない断片に**選ばれる**＝歩く exploit＝FAB が選んだ理由。
- 受け取り＝ガーディアンの**試練（選定）を越える**こと。戦闘とは限らない（§4）。

## 4. ガーディアン類型（宿主＝映像バリエの源）

各断片の宿主・試練は多彩。1断片＝1ダンジョン／1エピソードの核になりうる。

| 宿主 | 試練の性格（例） | 映像の核 |
|---|---|---|
| **人**（故人master・覚醒者・体制側の善人） | 対話・対決・資格の問答 | 群像・心理戦 |
| **動物**（異形・使い魔・群れ） | 本能・共生・信頼 | 生物的な迫力・情緒 |
| **架空AIキャラ**（深層の自我プログラム） | 論理・問答・矛盾突き | 非人間的・哲学的（FABと鏡像） |
| **自然・植物**（巨樹・苔・庭） | 時間・育成・忍耐（時間歪みと接続） | 静謐・幻想・スローライフ |
| **美術品**（像・絵・楽器） | 美・記録・再現 | 耽美・静物の非日常 |

- 宿主は**実在地ダンジョン**（[locations.md](locations.md)・[dungeon_design.md](dungeon_design.md)）に配置＝歪んだ駅/モール/団地/美術館/庭 等に多彩なガーディアン。

## 5. 集める意味＝origin への鍵（編集層 L4）

- 主人公の編集力は**自力で L0–L3**（[ability_progression.md](ability_progression.md)）。**L4＝01/量子＝origin** は、**Mythos 再構成が解禁鍵**。
- 断片を集める＝**世界の根幹へ近づく**＝大弧III の到達点。**集め切る＝最強（おれつえー）だが、10巻＝受容＝鍵を得てなお"改変を選ばない"**（テーマの頂点・[protagonist_core.md §3-6](protagonist_core.md)）。
- upstream（続編の種）＝Mythos が開く扉の先（[../plot/series_arc_draft.md](../plot/series_arc_draft.md)）。

## 6. 争奪（中心の macguffin）

| 勢力 | 断片をどう扱う |
|---|---|
| 無origin派（体制） | **封印・抹消**（origin 到達を防ぐ）。最大の妨害者 |
| 真主権者派 | 神/admin の復権の鍵として**渇望** |
| 原契約派 | 分散主権回復の証として**欲する**（だが中心化の誘惑も） |
| 上位世界派 | **upstream への鍵**として最重要視（10巻で正しさが露見） |
| **FAB** | **再merge を望み、主人公に集めさせる**。完成＝目的達成＝主人公は手段（7巻裏切り） |
