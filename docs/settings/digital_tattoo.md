## 1. 位置づけ：能力装備の主役

- 本作は能力戦。**武器・防具・道具という"持ち物"レイヤを、タトゥーが置き換える**。
  - 攻撃＝攻撃タトゥー／防御＝防御タトゥー（例：試作《軒下》＝防御タトゥー相当）／補助・移動・観測＝補助タトゥー。
- 起動＝そのタトゥー（関数）を呼ぶ。**技名は自由命名**（git 語を出さない、§writing_style）。
- **形は"力の象徴"（重要）**：タトゥーは剣・鎧・獣・道具などの姿を取りうるが、それは**能力の表現（象徴）であって、現実の金属の剣ではない**。剣型でも、中身は炎・巻き戻し・防御など**中の関数次第**。
  - だから**強弱は刃の鋭さでなく能力層（容量・literacy・相性・レア度）で決まる**＝「魔法 vs 剣」も同じ土俵の能力戦として成立する。
  - 姿は**使い手のあざ／自己像を映す**（キャラ個性。Accel World のアバターがトラウマから生まれるのと同じ発想）。剣戟・重装・ガジェット等の多様な"見た目"を、同じ能力層の上で出せる（映像映え）。
- 「装備を持ち替える」のではなく「**どのタトゥーを彫り、どう組むか**」が戦闘準備の本体。

## 1.1 タトゥーの正体＝圧縮されたコード **[確定方向]**

- **タトゥー＝情報が圧縮されたコード（関数/クラス/ライブラリ）の一種**。意匠（見た目の模様）＝**そのコードを人が彫れる形に圧縮・エンコードした表現**（QR/アーカイブ/ビルド成果物のイメージ）。git は表に出さず、在野では「模様」に見える。
- これで既存設定が一本に繋がる（下表）。

| 概念 | 圧縮コードとしての意味 |
|---|---|
| **現実化**（§5-2） | 圧縮コードを使い手が**展開・実行（解凍）**すること。引き出せる深さ＝**展開・実行できる力量**（literacy/容量/編集層） |
| **容量**（§6・stage §4） | 展開後の**サイズ＋裏依存**。圧縮率が高い（巧いコード）ほど軽い。dedupe/vendoring＝圧縮最適化 |
| **看破**（§6） | 熟練者は**意匠から中身（コード）を読み解く**＝literacy。手札がバレる理由 |
| **レア度**（§4） | 圧縮された情報の**質・希少性**。複製不可＝**展開鍵が master に bound** |
| **呪い**（供給網汚染） | 圧縮の中に紛れ込んだ**悪意あるコード**（読めない＝危険） |
| **あざ**（§2） | **最も濃く"自然圧縮"された署名**（生まれつき・複製不可・展開困難な領域を含む） |

- **彫師**（§8.5）＝この圧縮コードを身体に正しく書き込み、相性（展開可否）を診断する職人。

## 2. あざ と デジタルタトゥー（コピー可能性スペクトラム）

| | あざ（自然） | デジタルタトゥー（人工） |
|---|---|---|
| 出自 | 生まれつき・repo から自然発生 | 後天的に彫る |
| 複製 | **不可**（唯一無二） | コモンは可／レアは難／一部**複製不可** |
| 入手 | 不可（運命） | ダンジョン/自作/彫師 |
| 制御 | 制御不能 | 設計・常設できる |
| 永続性 | 一生（消えない） | 彫れば append-only で消えにくい |

> **コピー可能性の頂点**：コモンタトゥー → レア → 複製不可タトゥー → **生まれつきのあざ**。
> 主人公のあざは頂点＝**彫り取れない力**。ゆえに各勢力は力でなく**本人を確保**しようとする＝中心的な争奪の動機。

## 3. 出自（3系統）

| 出自 | 中身 | 性質 |
|---|---|---|
| **自作で彫る** | 自分で書いた関数/クラスを彫る | 軽い・所有・理解済み。要 literacy。最も尊ばれる |
| **ダンジョン入手** | 攻略報酬＝死者 master の repo の署名技タトゥー | レア度＝master の格／相性＝作風適合／**一部は複製不可（master 署名に bound）** |
| **彫師に彫ってもらう（import）** | 他人のライブラリを彫る | 即戦力だがブラックボックス＝**裏依存・供給網汚染（呪い）リスク** |

- **彫師（ほりし）**＝コードを他者の身体に deploy する職業。コモンは彫れるが、複製不可タトゥーは彫れない（ダンジョンが本人に直接 bind する）。
- ダンジョン徒弟(§8.5)との関係：**訓練して自分の関数として再実装＝自作タトゥー**／**master の技を直接授かる＝bound タトゥー（複製不可）**、の二択。

## 3.5 身体型 と 外付け（装置）型 **[確定方向]**

タトゥーを**どこに刻むか**で二系統。武器・防具・道具・中継のいずれも両型がありうる。

| | **身体タトゥー** | **外付け（装置）タトゥー** |
|---|---|---|
| 刻む先 | 身体＝repo | **物体（得物・防具・道具・ヘッドホン等）** |
| 着脱・譲渡 | 不可（一生背負う） | **着脱・譲渡・売買・継承できる** |
| 紛失・奪取 | されない | **奪われる・折られる・すり替えられる** |
| 監視 | 登録なら追跡 | **野良(off-chain)にしやすい＝監視の外** |
| 永続性 | append-only で消えない | 物体に依存（壊れる/失う） |
| 向く用途 | 能力の本体・常時携行 | **得物・道具・インターフェース・中継** |

- **同じ「剣」でも二型**：①**身体型**＝腕から青い線が具現する剣（試作「剣のひと」の少年型・常に手元・奪えない）／②**外付け型**＝実際の得物（刀身・柄）に刻んだ剣（**奪える・折れる・売れる・継げる**）。防具・道具も同様。
- **経済との接続**（§4.5/§8）：市場で**売買・継承されるのは主に外付け型**（身体タトゥーは譲渡不可）。登録品＝来歴つき／野良＝off-chain・**バックドアのすり替え**リスク。
- **ドラマの種**：外付けは**奪取・紛失・破壊**が物語になる（得物を折られる＝戦力喪失、形見の品を奪われる 等）。便利だが脆い。
- **コピー可能性スペクトラム上の位置**（§2）：**外付け＝最も奪いやすい端**／あざ＝**自然・身体・複製不可＝最も奪えない端**。

### 3.5-1. 主人公のヘッドホン（外付けタトゥーの実例）

- ナギの白いヘッドホンには**外付けタトゥーが埋め込まれている**＝**FABの接続アンカー（off-chain 中継）**。FABは主人公の身体（repo）に書き込まず、この装置を介して耳元に届く＝「ヘッドホン越しの声」(D3, [fab_persona.md](fab_persona.md))の機構。**監視の外**で繋がれる。
- **実は失った親の仕込み**（[protagonist_core.md](protagonist_core.md)。由来は中盤開示・[../series/disclosure_ledger.md](../series/disclosure_ledger.md)）。ヘッドホン＝親の品＋親が遺した外付けタトゥー＝FABの通り道。**喪失・FAB・タトゥー体系が一つの小道具に集約**。
- **弱点**：奪われ/壊れれば**FAB回線が断たれ、親の欠片も失う**。
- Ep1 では「昔から使ってる音声デバイス」程度に伏せる（親由来・FABアンカーの自覚は後）。

## 4. レア度

| 階梯 | 在野の呼び名 | 複製 | 入手 | 例 |
|---|---|---|---|---|
| コモン | **ありもの／量産もの** | 自由 | 市場・浅層ダンジョン | 基本攻撃/防御/灯り |
| レア | **上もの（じょうもの）** | 難（要・高位彫師） | 中〜深層 | 特化した強技 |
| **複製不可（ユニーク）** | **一点もの／形見（かたみ）** | **不可（bound）** | 特定ダンジョン攻略のみ | 死者 master の唯一の絶技 |

- **在野呼称はライトに**（git 語を出さない）：「ありもの」「上もの」「一点もの」。複製不可は死者 master 由来ゆえ **「形見」** とも呼ばれ、情緒・継承のニュアンスが出る（強奪＝形見を奪う、の重さ）。
- **複製不可の理由（作者側）**：①master 署名に bound（その人にしか verify しない）②fork/copy を禁じる license ③ライブラリでなく master の repo 断片そのもの（複製できない遺産）。→ **唯一性＝争奪・盗用・継承の物語装置**。

## 4.5 ブロックチェーン管理 と 野良（来歴・真贋・監視） **[確定方向]**

タトゥーには、**ブロックチェーン（分散台帳）で管理された正規品**と、**野良（off-chain・無登録）**が混在する。

| | ブロックチェーン管理 | 野良（off-chain） |
|---|---|---|
| 真贋・来歴 | 証明可能（誰が作り誰が持つか追跡可） | 不明・自己申告 |
| 複製不可の保証 | チェーンが唯一性を担保（NFT的） | 保証なし（本物か偽物か不明） |
| 監視 | **追跡される**（当局/Glasswing に可視） | **追跡されない**（パノプティコンの外） |
| 入手 | 正規市場・公認彫師・登録ダンジョン | 闇市・無登録彫師・未踏ダンジョン・自作未登録 |
| リスク | 体制に把握される（自由と引き換え） | 裏依存・呪い・偽物のリスク高（要 literacy） |
| 立場 | 合法・身分証・正規戦力 | 覚醒者/アウトロー向け・隠れる力 |

- **覚醒者・主人公は野良を好む**：登録＝追跡だから。deadzone で戦うのと同じ「見られず持つ」論理(battle §1-2)。ただし野良は**読めない＝危険**（偽物・呪い・未知依存）＝literacy が生死を分ける。
- **registry を誰が運営するか＝新たな権力**（彫師ギルド／Glasswing／学派／クラウド）。正規化＝統制の道具にもなる（§8.5 強制 sync の烙印と地続き）。
- **深層の伏線（皮肉）**：人類はタトゥー管理に "分散台帳" を発明したのに、**自分たちが遥かに巨大な分散台帳＝世界＝git の中にいることに気づいていない**。ブロックチェーンは**深層 git の影／再発明**。覚醒者がこの符合に気づくのが、深層へ落ちる入口の一つになりうる。
- **主人公のあざ**＝自然物ゆえ**チェーンに載らない＝登録も証明も超越**＝「台帳の外の本物」。複製不可をチェーンが担保する世界で、あざだけが別格。

## 5. 相性（compatibility）――インフレ回避の核

- **タトゥー同士**：依存が噛み合えばコンボ（合成で強化）、競合すれば同時起動不可（battle_system §2）。
- **装着者との相性**：タトゥーの依存グラフが**自分のあざ/repo の性質**に合えば軽い、合わなければ重い（裏依存が容量を食う、§4）。
- → **最強は"最高レア"ではなく"最も相性よく組んだ者"**。レジェンダリー1枚 ＜ 相性抜群のコモン＋コンボ。

### 5-1. 相性判定の4分類（規則化）

タトゥーが要求する**裏依存（前提）**が、装着者の **あざ/repo に既にあるか**で決まる：

| 分類 | 状態 | 結果 | 在野の言い方 |
|---|---|---|---|
| **① 噛み合う** | 必要な依存を装着者が既に持つ（再利用） | **軽い**・コンボ可 | 「手に馴染む」 |
| **② 要追加** | 不足依存を丸ごと抱える | 使えるが**重い**（容量圧迫） | 「重い」「持て余す」 |
| **③ 競合** | 既存の技と同じ枠を奪い合う | **同時起動不可**・干渉 | 「ケンカする」 |
| **④ 拒絶** | あざ/repo の気質と矛盾 | **拒絶反応・呪い化**の危険 | 「身体が受けつけない」 |

- **彫師の腕＝この4分類を事前診断する目利き**（§8.5）。診断を誤れば②③④で容量超過・干渉・呪い。
- **野良は無診断のギャンブル**：来歴不明（§4.5）＋相性未診断＝literacy で自衛するしかない。
- 主人公の**ハーフ＝二系譜 repo**ゆえ相性が**ねじれる**：普通は④拒絶の技が①噛み合いになり、逆に皆が①の定番技が③④になる。**他人に扱えない技を使えるが、定番が使えない**＝固有戦術かつ孤立の理由。

### 5-2. 現実化と選定（使い手の力量・資格）**[確定方向]**

- **タトゥーは"潜在"。引き出せる深さ＝使い手の力量**（literacy・容量・編集層、[ability_progression.md](ability_progression.md)）。
  - **レア度＝潜在の上限／実効＝使い手の力量**。**弱者が強タトゥーを持っても浅くしか現実化しない**（＝拾っただけでは勝てない・loot バランス・インフレ回避）。達人は同じタトゥーから深い層を引き出す。
- **使い手を選ぶタトゥー（選定タトゥー）**：**資格（master 署名・あざ適合・相性①）が無いと現実化しない／拒絶・呪い化（④）**。
  - **形見（複製不可・bound）＝選定の極**（master に縛られ、継承資格が要る、§8.5-5）。
  - 野良で拾った絶技が「使い手を選ぶ」＝**使えず封印 or 蝕まれる**（強奪してもおれつえーにならない）。
- **主人公のねじれ（固有性）**：あざ＝異常適合ゆえ、**普通は誰も選ばれない深層・禁忌タトゥーに選ばれる**一方、**皆が使う定番に拒まれる**（§5-1 と一貫）。
- 演出：選定をくぐった時の高揚／資格なき者が手を出して蝕まれる悲劇。

## 6. 装着・容量・可視性

- 装着数・強度は**容量＝サイズ予算＋裏依存**(§4)に縛られる。盛りすぎ＝crash（記憶喪失＋現実巻き戻し）。
- **placement（部位・意匠）がキャラの個性**＝ビジュアル/グッズ/短尺動画映え。
- **可視ゆえのトレードオフ**：熟練者は相手のタトゥーを**読んで手札を割る**（看破）。**見せる力（タトゥー）vs 隠す力（主人公の natural あざ）**。
- **彫る＝永続**：毎戦スワップではなく、生き様としての積み上げ。停止・上重ねは可、真の消去は困難（append-only）。だからビルド選択が重い。

## 7. 戦闘運用（型）

| 型 | 役割 | 例 |
|---|---|---|
| 攻撃タトゥー | 武器の代替 | 斬撃・射出・範囲 |
| 防御タトゥー | 防具の代替 | 障壁・受け流し（《軒下》型） |
| 補助タトゥー | 道具の代替 | 観測・移動・治療・解析（観測戦＝battle §5 と連動） |

- 起動は自由命名の技名で（git 語禁止）。コンボ＝複数タトゥーの連結。
- 心意級（force-write、battle §6）は**タトゥーを超える**領域＝あざ/異常 commit 側の禁忌。

## 8. 経済・社会

- **彫師ギルド**・タトゥー市場・レア度相場＝ダンジョン経済(§3)の中核。
- **複製不可タトゥー**は売買できず（bound）→ **争奪・強奪・継承（死者からの相続）**が紛争を生む。
- タトゥー＝身分証/所属証にもなる（§anomaly_mark §0.5：権限・署名・**強制 sync の烙印**）。社会階層：自作できる者 ＞ 良い彫師に彫れる者 ＞ 安物 import で済ます者。

## 8.5 彫師（ほりし）とギルド

> 自作できない大多数は、**彫師に頼んで技を身体に定着させる**。彫師＝他人の身体（repo）にタトゥーを deploy する職人であり、**相性（§5-1）を読む目利き**。

### 8.5-1. 彫師の仕事

- **技タトゥーを客の身体に彫り、定着させる**＝import/install の代行。彫る＝commit＝**append-only で消えない**（§0）ゆえ、施術は不可逆で重い責任を負う。
- 本領は**相性診断**：客のあざ/repo を読み、その技が ①噛み合う〜④拒絶（§5-1）のどれかを見極め、「載る/載らない・軽い/重い」を判定する。**名彫師＝相性診断士**。
- 複製不可（形見）は彫師でも**複製できない**（master 本人に bound）。ただし**ダンジョン産の形見タトゥーを身体に定着させる施術・継承儀式は介助できる**（死者→生者の移譲の作法）。

### 8.5-2. 技術階梯

| 階梯 | できること |
|---|---|
| **見習い** | ありもの（コモン）を彫る。相性は定番パターンのみ判別 |
| **一人前** | 上もの（レア）を彫る。①〜④の相性診断ができる |
| **名彫師** | 危険な依存・競合を捌く。形見の定着儀式・継承を担う。**呪い（供給網汚染）を見抜く/解く**腕 |

### 8.5-3. ギルド構造（登録 vs 野良）――§4.5 と地続き

| | 登録ギルド（公認彫師） | 野良彫師 |
|---|---|---|
| 立場 | 免許・身分保証・ブロックチェーン管理 | 無認可・闇市 |
| 扱う品 | 正規・来歴つき・追跡される | 唯一物・呪い物・未登録も扱う |
| 監視 | **当局/Glasswing に可視**（誰が何を彫ったか残る） | **監視の外**（だが自己責任） |
| 安全 | 相性診断つき・呪い混入を規制 | 無診断もありうる・**バックドアを彫られる危険** |
| 客 | 一般市民・正規戦力 | 覚醒者・アウトロー・隠れたい者 |

- **registry を運営する者＝権力**（§4.5）。ギルドは「**誰に何を彫ってよいか**」を免許制で統制＝体制側の締め付け装置になりうる（強制 sync の烙印と地続き、§8）。
- **悪徳彫師の闇**：彫る＝不可逆ゆえ、客の知らない**裏依存（バックドア）や呪いを彫り込む**犯罪が成立。野良の魅力（自由・唯一物）と恐怖（不可逆の罠）が表裏。

### 8.5-4. 主人公との関係

- 主人公は**自作できる＝彫師いらず**＋**あざ＝彫り取れない複製不可**。→ ギルドの権力構造・経済の**外側**に立つ＝**目をつけられる異物**（免許制の外で力を持つ者）。
- 野良彫師は主人公にとって**情報源かつ危険源**：唯一物や深層の噂を持つが、あざを解析・利用しようとする者もいる（series_arc の敵/協力者の供給源）。

### 8.5-5. 形見（複製不可）の継承――相続と強奪

形見は売買も複製もできない（master に bound）。生者が得る唯一の道は**ダンジョン（master の残存 repo）が攻略者本人に直接 bind する**こと。その定着には作法がある。

| 継承の型 | 何が起きるか | 載り方 |
|---|---|---|
| **正統相続** | master が生前「継ぐ者」を署名で許可 → ダンジョンが資格を verify | **素直に bind**（①噛み合う・軽い）。名彫師が定着を介助し安定 |
| **強奪（無資格攻略）** | 許可なき者が攻略しても bind は起きる | **ねじれて載る**（④拒絶に傾く・呪い化の危険）。使えても**蝕む**＝盗品の重さ |

- **不可逆**：一度 bind した形見は本人が死ぬまで外せない（append-only）。**形見を負う＝故人の生き様を背負う**＝技と一緒に記憶・無念・癖まで載る（戦闘スタイルが故人に似る等の演出）。
- **死後の再継承**：形見の持ち主が死ぬと repo は main へ merge（§stage 死＝合流）。技は再びダンジョン化した残骸から次の資格者へ──の循環。**形見＝世代を越える絆／呪い**。
- **あざとの対比**：あざ＝**生まれつき bound の究極の形見**（起源からの相続を超えた自然物）。彫師にも継承儀式にも依らず最初からそこにある＝別格（§2 頂点）。
- 物語装置：形見争奪戦／**継承資格の問答**（誰が継ぐにふさわしいか）／奪った者が蝕まれる悲劇／故人の癖が技に滲む情緒。

## 9. 主人公との接続

- 主人公の**あざ＝究極の複製不可**＝彫り取れない＝**人ごと狙われる**（中心的争奪、series_arc）。
- ハーフ＝二系譜ゆえの**異常な相性**で、他人に扱えないタトゥーを使える/その逆＝固有性。
- 歩く exploit ＝本来彫れない領域に彫れる（自作で禁忌タトゥーを書けてしまう）可能性。

## 10. 設計原則チェック

- **インフレ回避**：強さ＝レア度でなく**相性・literacy・コンボ**。レジェンダリー偏重を否定。✅
- **git を表に出さない**：技名・在野呼称は平易/自由。彫る機構は作者側のみ。✅
- **ライト文体**：在野では「タトゥー」「あざ」「彫師」等わかりやすく。✅
- **append-only 整合**：彫る＝消えない＝ビルドが重い。✅
