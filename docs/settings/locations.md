## 1. 表層と深層の関係（移動・アクセス）

- 世界は**二層**（[../stage_device_design.md §5.3](../stage_device_design.md)）。表層＝公知の現実（2030年以降、ID・タトゥー・ダンジョンは常識）。深層＝偽origin・git機構（覚醒者のみ）。
- **ダンジョン＝現実の特定地点に重なる"深層"の異相**。別世界にログインするのではない。
- **アクセルワールドとの差別化（重要）**：AW＝「意識だけ加速し身体は現実で安全に固まる」。本作は**身体ごと潜る・身体に代償・加速は局所技**で別物にする（[battle_system.md §1-3](battle_system.md)）。
- **アクセス（移動）＝ダイブ**：
  1. 現実の街で、ダンジョンの**入口（実在の座標）**まで物理的に行く（電車・徒歩）。
  2. 入口で**潜行（ダイブ）**＝**身体ごと深層インスタンスに入る**（意識だけ置き去りにしない）。現実側＝「ダンジョンに入った」＝デッドゾーン（監視外）。**公共の場で身体が固まる絵にしない**。
  3. **帰還アンカー（戻り標）が入口に残る**：ダイブ時、入口の現実座標に**空間マーキング（タトゥー状の紋様＝圧縮コードの一種）**が生成される。
     - **現実側から物理干渉できない**（壊せない・動かせない・塞げない。深層に根ざすため）。
     - だが**覚醒者・監視には探知できる**＝「誰がいつ潜ったか・どこへ戻るか」が読める → **追跡・待ち伏せ**が成立。
     - **帰還時に使う**：ダイバーはこのアンカーを通って表層へ戻る＝**帰還地点が割れている**。
  4. 深層の**消耗・傷は現実の身体に反映**（高結合）。安全な観戦アバターではない。撤退/攻略で帰還、**crash＝記憶喪失＋現実巻き戻し**（深層では致死もありうる、battle §4-2）。
- **帰還の瞬間が最も脆い**：消耗して戻るところを待ち伏せされる＝戦術ドラマ（アンカーを守る・囮アンカー・観測戦 §5 と接続）。
- **アンカー＝インスタンス（ブランチ）へのハンドル**：ダイブ＝インスタンス生成。**同じアンカーをくぐれば合流**（複数人で訓練/協力）、共有しなければ別インスタンス（出会えない）。敵がアンカーを掴めば**乱入**、強者は**強制的に自分のインスタンスへ引き込む**（逃げ場なし・第1話で悠が引き込まれた弱い版）。詳細＝[dungeon_design.md §1.5-5](dungeon_design.md)。
- **登録ダンジョン**は入口・アンカーが公知（建物・門）。**野良/未踏**は入口が隠れているが、**アンカーは残る＝痕跡**（FAB の off-chain 技で薄める駆け引き）。

## 2. 自宅

- ナギの日本での住まい＝**横浜・元町の坂のふもとの古い下宿**（みなとみらい線・元町中華街駅寄り）。洋館街の空気・ハーフ留学生に馴染む。**[未決]同居者**＝大家／ホスト／身内（protagonist_core と連動）。
- **移動の動線（自然さ）**：自宅（元町ふもと）→坂を上れば学校（山手）→放課後は**みなとみらい線一本でみなとみらい駅**。元町中華街〜みなとみらいは直通数分で、山手とみなとみらいの行き来が無理なく繋がる。
- **機能**：日常・休息・**睡眠＝技の定着**（battle §3.5）。新天地で得る温もり（日本食・畳・人の親切）＝**FABの削れで失われていく対象**（fab §4.5）。
- **ビジュアル**：畳・障子・台所の湯気・縁側。生活の温度。留学生の新鮮な目（日本文化）。
- **対比**：温かい日常 ⇔ 深層の孤独・喪失。ここを丁寧に描くほど喪失が効く。

## 3. 学校

- 留学先の学校＝**横浜・山手駅をモチーフ**（煉瓦の校舎・鐘楼・洋館街）。**表層世界の中心**。同年代の人間関係、日本での"外来者"としての立ち位置（容姿が欧米寄り＝目立つ）。
- **機能**：覚醒の隠蔽（あざを前髪で隠す・深層を秘密にする日常）、コメディと対比、仲間/ライバルの登場の場（**剣の少年＝同級生にできる**＝監視と友情の二重）。
- **ビジュアル**：現代日本の学校。日常のテンポ（editor-tsukikage 的な群像・掛け合い）。
- **対比**：放課後にダンジョンへ＝日常から深層への落差。

## 4. 訓練

- 特訓シーンを量産する場（著者方針＝戦闘・特訓多め）。三段階：
  | 型 | 場所 | 中身 |
  |---|---|---|
  | 基礎 | 自宅の庭・空き地・河川敷 | 技の試行（《軒下》《晴れ間》）→睡眠で定着（試作「ひと晩寝かせる」） |
  | 実戦 | **浅層ダンジョン／野良の安全な層** | 弱い敵で技を試す・失敗する（naro 原則＝失敗を描く） |
  | 師事 | 野良彫師・先達の所 | 相性診断・タトゥー・知識（digital_tattoo §8.5） |
- **ビジュアル**：日常の場所＋淡い青い線（能力モチーフ）。実戦は深層の歪んだ実在地。

## 5. ダンジョン（→ 詳細 [dungeon_design.md](dungeon_design.md)）

- **実在地（GoogleMap）を歪めた深層インスタンス**。駅・モール・団地・学校・川辺・地下街などの**実在の場所の歪んだ版**。
- **実在地モチーフ＝横浜**：第1ダンジョン＝**みなとみらい駅**（赤いエスカレータの大吹き抜けアトリウム＝歪ませ映え）。以後も横浜の名所（みなとみらい地区・地下街等）を歪めて舞台化。参考画像＝[../../media/reference/](../../media/reference/)。
- **インスタンス性**：入るたび別生成。**故人masterの固有現象＝そのダンジョンの"癖"（同一性）**は保ちつつ、層構造・天候・時間の流れ・歪み方は**ランダム**。同じ環境は稀。
- **時間歪み**：内部は加速。さらにダンジョン/回ごとに時間の流れ方が違う（速い/淀む/巻き戻る）＝固有現象の一種。
- 詳細・生成規則・移動は [dungeon_design.md](dungeon_design.md)。

## 6. 映像化方針（再作成の狙い）

- **実在地ベースで撮る**：抽象空間でなく「歪んだ実在の場所」にすることで、背景が具体的・多様・記憶に残る（旧試作「記録の雨の三層」が弱かった点の修正）。
- 表層（学校・自宅の生活感）と深層（歪んだ街の非日常）の**落差**を1シーンに入れる。
- 同じ場所が毎回違う姿になる＝**ビジュアルのバリエーションが無限**（短尺量産に有利）。
