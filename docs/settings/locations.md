## 1. 表層と深層の関係（移動・アクセス）

- 世界は**二層**（[../stage_device_design.md §5.3](../stage_device_design.md)）。表層＝公知の現実（2030年以降、ID・タトゥー・ダンジョンは常識）。深層＝偽origin・git機構（覚醒者のみ）。
- **ダンジョン＝現実の特定地点に重なる"深層"の異相**。別世界にログインするのではない。
- **アクセルワールドとの差別化（重要）**：AW＝「意識だけ加速し身体は現実で安全に固まる」。本作は**身体ごと潜る・身体に代償・加速は局所技**で別物にする（[battle_system.md §1-3](battle_system.md)）。
- **アクセス（移動）＝ダイブ**：
  1. 現実の街で、ダンジョンの**入口（実在の座標）**まで物理的に行く（電車・徒歩）。
  2. 入口で**潜行（ダイブ）**＝**身体ごと深層インスタンスに入る**（意識だけ置き去りにしない）。現実側＝「ダンジョンに入った」＝デッドゾーン（監視外）。**公共の場で身体が固まる絵にしない**。
  3. **帰還アンカー（戻り標）が入口に残る**：ダイブ時、入口の現実座標に**空間マーキング（タトゥー状の紋様＝圧縮コードの一種）**が生成される。
     - **現実側から物理干渉できない**（壊せない・動かせない・塞げない。深層に根ざすため）。
     - だが**覚醒者・監視には探知できる**＝「誰がいつ潜ったか・どこへ戻るか」が読める → **追跡・待ち伏せ**が成立。
     - **帰還時に使う**：ダイバーはこのアンカーを通って表層へ戻る＝**帰還地点が割れている**。
  4. 深層の**消耗・傷は現実の身体に反映**（高結合）。安全な観戦アバターではない。撤退/攻略で帰還、**crash＝記憶喪失＋現実巻き戻し**（深層では致死もありうる、battle §4-2）。
- **帰還の瞬間が最も脆い**：消耗して戻るところを待ち伏せされる＝戦術ドラマ（アンカーを守る・囮アンカー・観測戦 §5 と接続）。
- **アンカー＝インスタンス（ブランチ）へのハンドル**：ダイブ＝インスタンス生成。**同じアンカーをくぐれば合流**（複数人で訓練/協力）、共有しなければ別インスタンス（出会えない）。敵がアンカーを掴めば**乱入**、強者は**強制的に自分のインスタンスへ引き込む**（逃げ場なし・第1話で悠が引き込まれた弱い版）。詳細＝[dungeon_design.md §1.5-5](dungeon_design.md)。
- **登録ダンジョン**は入口・アンカーが公知（建物・門）。**野良/未踏**は入口が隠れているが、**アンカーは残る＝痕跡**（FAB の off-chain 技で薄める駆け引き）。

## 2. 自宅

- ナギの日本での住まい＝**横浜・本牧の古い下宿**（**鉄道駅が無い静かな住宅地**・海が近い・坂と緑）。駅が無いぶんバス通学＝外来者の不便もにじむ。**足元の深層に幻の地下鉄駅**が眠る（§4.5）。**[未決]同居者**＝大家／ホスト／身内（protagonist_core と連動）。
- **移動の動線（正確に）**：自宅（**本牧**・中区本牧町・駅無し・山手の東）→**バス**で学校（**JR山手駅**・根岸線）→放課後は**山手駅から根岸線で桜木町、運河沿いを徒歩でコスモワールド**（第1ダンジョン＝**裏手の非常口**）。本牧→山手駅は西へ、山手駅→桜木町は根岸線で数駅、桜木町→コスモワールドは徒歩数分＝学校帰りの寄り道として自然。※元町・中華街駅／みなとみらい線ではない。
- **機能**：日常・休息・**睡眠＝技の定着**（battle §3.5）。新天地で得る温もり（日本食・畳・人の親切）＝**FABの削れで失われていく対象**（fab §4.5）。
- **ビジュアル**：畳・障子・台所の湯気・縁側。生活の温度。留学生の新鮮な目（日本文化）。
- **対比**：温かい日常 ⇔ 深層の孤独・喪失。ここを丁寧に描くほど喪失が効く。

## 3. 学校

- 留学先の学校＝**カトリック系ミッション男子校**。**モチーフ＝聖光学院中学校・高等学校**（**JR山手駅最寄り**・著者地図指定 2026-06-14）。チャペル（鐘の塔）・煉瓦の趣＝山手の洋の器。**表層世界の中心**。同年代の人間関係、日本での"外来者"としての立ち位置（容姿が欧米寄り＝目立つ）。**作中校名＝「聖稜学院」**（著者確定 2026-06-14・モチーフ化＝仮名）。
- **山手＝旧外国人居留地の丘**：西洋館とカトリック・ミッション校が集まる土地。**欧米寄りハーフの主人公の舞台に最適**。主人公校（聖光モチーフ・男子）↔ ヒロイン校（横浜雙葉モチーフ・女子）が**同じ"洋の器"の丘に並ぶ**（[heroine_core.md §1](heroine_core.md)）。
- **機能**：覚醒の隠蔽（あざを前髪で隠す・深層を秘密にする日常）、コメディと対比、仲間/ライバルの登場の場。
- **ヒロイン校＝山手のミッション女子校（弓道部）**：**モチーフ＝横浜雙葉中学・高等学校**（山手の丘・カトリック女子校。横浜女学院ではない）。**弓道部のヒロイン（弓道狙撃・登録監視・見逃す葛藤型）**の所属（[heroine_core.md](heroine_core.md)・第2話）。「洋の器（ミッション校）に和の芯（弓道）」＝ハーフ主人公の鏡像。**作中校名＝「山手聖二葉女学院」**（著者確定 2026-06-14・モチーフ化＝仮名）。ヒロイン＝**篠宮 澪**（[heroine_core.md](heroine_core.md)）。両校は同じ丘だが**隣接ではない**（雙葉＝山手町北／聖光＝山手駅南）ため、ナギとの遭遇は**校門でなく放課後の動線・監視で起こす**。※剣の少年＝同級生案は**後巻の近接ライバル候補（聖光モチーフ男子校）として温存**。
- **ビジュアル**：現代日本の学校。日常のテンポ（editor-tsukikage 的な群像・掛け合い）。
- **対比**：放課後にダンジョンへ＝日常から深層への落差。

## 4. 訓練

- 特訓シーンを量産する場（著者方針＝戦闘・特訓多め）。三段階：
  | 型 | 場所 | 中身 |
  |---|---|---|
  | 基礎 | 自宅（本牧）の庭・空き地 | 技の試行（《アンブレラ》《サンブレイク》）→睡眠で定着（試作「ひと晩寝かせる」） |
  | 実戦・拠点 | **本牧の幻の地下鉄駅**（§4.5） | 安全寄りの隠れ家で実戦練習。弱い敵で技を試す・失敗する（naro 原則＝失敗を描く） |
  | 師事 | 野良彫師・先達の所 | 相性診断・タトゥー・知識（digital_tattoo §8.5） |
- **ビジュアル**：日常の場所＋淡い青い線（能力モチーフ）。実戦は深層の歪んだ実在地。

### 4.5 本牧の幻の地下鉄駅（訓練拠点・隠れ家）

- **本牧＝表層には鉄道駅が無い土地**（実在の特徴。過去に未成・幻の地下鉄延伸計画があった）。**なのに深層には、その"在ったかもしれない駅"＝幻の地下鉄駅が在る**。表層／深層のズレの象徴。
- **固有現象＝「終わらない終着駅／淀む終電」**（[canon_log C-029](../series/canon_log.md)）：停車車両は永遠に発車せず、時間が静かに淀む＝**誰かを待ち続けて発てなかった人の記憶**（master の repo のにじみ・素性はB1では気配のみ＝**ナギの喪失の鏡**）。だから穏やかで安全＝隠れ家向き。発車標・路線図は無文字（凍結）。native＝待ち続ける乗客の影。
- **拠点化＝シードで固定**（第3話[ep3_plot.md](../plot/ep3_plot.md)・機構更新 2026-06-16）：入るたび中身が変わるインスタンスから**当たりの一本を引き当て、その核に到達して"シード値"を取得＝固定**＝流動しない**自分だけのインスタンス**に（[dungeon §1.5-6](dungeon_design.md)）。**シード再訪は戻り標を残さない＝外から辿れない＝真のデッドゾーン**（戻り標は時限で薄れる）。＝**一発で当たりを引き、その核に到達してシードを読み取った**想定。以後、停車車両とホームを居室化した**温かい隠れ家**（ベッド・ソファ・キッチン・ランタン・植栽＝状態保持）。**第2話＝冷たい廃駅（発見）→第3話＝温かい隠れ家（確立）**。背景アセット＝[honmoku_station_hideout_night_v1.png](../../media/common/backgrounds/honmoku-phantom-station-hideout/outputs/honmoku_station_hideout_night_v1.png)（確定）。
- **訓練拠点に向く理由**：①現実に存在しない＝**監視の外**（あざを探知されず練習できる）②安全寄り（制御された固有現象・弱い個体）③**アンカー共有で仲間と合流**して訓練できる（dungeon §1.5-5）④集合・休息・作戦に使える隠れ家。
- **発見＝寄り道**：ナギは**本牧に住んでいる**。深層の異常に惹かれる性質（歩く exploit）ゆえ、気になって寄り道し、現実には無いはずの駅の気配＝幻の駅の入り口を見つけてしまう（コスモワールドの非常口へ引かれたのと同じ"惹かれ"）。確立は第2話以降。
- **未成/廃ゆえの不安定さ＝イレギュラーの温床**：幻の駅は**安定化されたことのない未成インスタンス**。普段は安全寄りの隠れ家だが、**固有現象が暴走するイレギュラー**がありうる（第2話＝ヒロインとの対峙中に暴走に二人で巻き込まれる＝[heroine_core.md §1.5](heroine_core.md)）。イレギュラー＝**点の無い混沌**（座標と瞬間が溶けて無数に重なる）＝**一点を射抜く弓（澪）と相性最悪**／**面（起きた出来事）に触れるナギ**が継ぎ目を掴む＝点 vs 面。一過性の事象で、抜けた先の"嵐の目"が拠点になる。

## 5. ダンジョン（→ 詳細 [dungeon_design.md](dungeon_design.md)）

- **実在地（GoogleMap）を歪めた深層インスタンス**。駅・モール・団地・学校・川辺・地下街などの**実在の場所の歪んだ版**。
- **第7話ダンジョン＝根岸森林公園＋旧競馬場の廃馬見所**（[canon_log C-037](../series/canon_log.md)）：崩れかけた三つの塔（蔦に呑まれた廃墟）と、馬が走った楕円の草道。**固有現象＝終わらないレースの歓声**（観客はいないのに歓声と見えない蹄だけが回り続ける＝走り終えたレースが終われない＝群衆の記憶・master のにじみ・素性はB1では気配のみ＝ナギの喪失の鏡。隠れ家「終わらない終着駅」§4.5 と同系の"待ち/終われない"motif）。**ここに道化の楽団（C-028）の署名が上書き侵入＝B1初の道化再来・気配のみ**（手回しオルガン→火の粉→森化）。根岸＝山手と本牧の中間（放課後の動線に乗る・引きの火の粉が本牧へ流れる動線が自然）。
- **実在地モチーフ＝横浜**：自宅＝**本牧**（中区本牧町・駅無し＋深層に幻の地下鉄駅＝訓練拠点 §4.5）／学校＝**JR山手駅**（根岸線・煉瓦校舎・鐘の塔）／第1ダンジョン＝**コスモワールドの非常口**（みなとみらいの大観覧車。閉園した遊園地に、誰も乗せず回り続ける観覧車。**固有現象＝記録の雨**。**native（本来の出現）＝ずぶ濡れの獣**だが、Ep1の遭遇は**「道化の楽団＝謎集団」が深層の異常（バグ）で上書き侵入**＝彼らの署名「**炎と森のカーニバル**」（森に呑まれた遊具・火の粉のような灯）がコスモワールドに染み出した上書き＝[canon_log C-028](../series/canon_log.md)。SEKAI NO OWARI〈炎と森のカーニバル〉着想。学校帰りに山手駅→根岸線→桜木町→徒歩）。以後も横浜の名所を歪めて舞台化。参考画像＝[../../media/book1/ep1/reference/](../../media/book1/ep1/reference/)。
- **インスタンス性**：入るたび別生成。**故人masterの固有現象＝そのダンジョンの"癖"（同一性）**は保ちつつ、層構造・天候・時間の流れ・歪み方は**ランダム**。同じ環境は稀。
- **時間歪み**：内部は加速。さらにダンジョン/回ごとに時間の流れ方が違う（速い/淀む/巻き戻る）＝固有現象の一種。
- 詳細・生成規則・移動は [dungeon_design.md](dungeon_design.md)。

## 6. 映像化方針（再作成の狙い）

- **実在地ベースで撮る**：抽象空間でなく「歪んだ実在の場所」にすることで、背景が具体的・多様・記憶に残る（旧試作「記録の雨の三層」が弱かった点の修正）。
- 表層（学校・自宅の生活感）と深層（歪んだ街の非日常）の**落差**を1シーンに入れる。
- 同じ場所が毎回違う姿になる＝**ビジュアルのバリエーションが無限**（短尺量産に有利）。
