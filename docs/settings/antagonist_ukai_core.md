## 0.5 容姿・口調・佇まい（確定）

| 項目 | 内容 |
|---|---|
| **外見** | 30代後半〜40代。中背・痩せ型でも肥えてもいない、**印象に残らない実務官の体躯**。暗色のスーツ（皺なし・体制の制服に近い）。短髪、整えた手元、磨いた靴。**顔は整っているが生気が薄く、笑っても目が動かない**。すれ違っても覚えにくい＝**"顔の見えない制度の手"の人間版**（道化の楽団＝骸骨化粧の派手さの対極＝C-028）。 |
| **深層の姿** | 深層でも装いはほぼ変わらない＝**積み上げを"力"でなく"権限"に振った者**（[digital_tattoo §6.5](digital_tattoo.md)）。装束が華美に外在化せず、**手元に観測の鵜（光の鳥／無数の小さな目）だけが灯る**。強さの誇示でなく、**見ていることの圧**で立つ。 |
| **口調** **[確定]** | 一人称＝**「私（わたし）」**。**事務的・平坦・敬語**。声を荒げない。説得でなく**通告**。例「確認させてください。昨日、あそこに入りましたね」「これは処罰ではありません。手続きです」「あなたが消えても、誰も困りません。それが、いちばん穏やかな結末です」。**相手の名を呼ばず「対象」「あなた」で済ます**＝人を個として見ない冷たさ。澪に対しても**かつての後輩を見るような、わずかな"既視感"だけを覗かせる**（芯の設計＝§1.5）。 |

## 1. なぜ「足取りの鵜飼＝観測者／射手型」なのか（設計の収束）

既存機構を**追加なしで束ねる**当面の敵：

- **第1話の最大の伏線「帰還の脆さ・戻り標」を脅威として人格化**：澪が「帰り道を射抜く」狙撃手なら、鵜飼は「**帰り道の残響そのものを辿って住処を割り出す**」追跡者。同じ伏線（[battle §1-3](battle_system.md)）の、攻撃でなく**追跡**側の顔。
- **撃破でなく出し抜く決着が機構的に必然**（[ARC-01 課題C](../plot/book1_structure.md) を解消）：本人が弱く観測優越がすべて＝**観測を盲目化すれば無力化できる**。致死を避けつつ"勝った"と言える決着が設計から導かれる（[§3](#3-出し抜くなかったことにさせない決着の機構)）。
- **点 vs 面の連携が完成形で噛む相手**：ナギ＝点（歩く exploit・デッドゾーンに隠れる）×澪＝面（監視網側の証言）でしか出し抜けない＝**二人の共闘が"勝利条件"そのもの**になる（[ep2 で立てた軸](../plot/ep2_plot.md)）。
- **"上"を伏せたまま体制の冷たさを体感させる**：末端ゆえ本人も全体像を知らない＝[disclosure](../series/disclosure_ledger.md) を破らずに「体制の体温のなさ」だけを先に届けられる。議長セン（8〜9巻）の予告編。
- **道化の楽団（C-028）との対極**：派手・無秩序・正体不明の侵入者（道化）に対し、鵜飼は**地味・秩序・正体は制度の手**。巻1の二種の脅威が「無秩序の異物」と「秩序の冷たさ」で対称をなす。

## 1.5 「澪の未来の影」＝鏡像としての芯

> 著者決定（2026-06-15）＝鵜飼の内側の動機を**「澪の未来の影」**として設計する。§1.5 暫定の「官僚的で冷たい＝議長セン末端版」を、**"そうなった理由"**で裏打ちする。

### 設計の核

鵜飼は、**かつての澪**である。
- 同じ「登録・監視」系の人間として、彼も若い頃**「見つけて・確かめて・報告する」職務と、目の前の個を見逃したい心**の板挟みに立った。
- だが彼は、その葛藤を**「手続き」で塗り潰すこと**を選んだ。何度も見逃しかけ、何度も呑み込み、やがて**何も感じなくなった**。いまの平坦さは、生まれつきの冷たさでなく、**摩耗の果て**。
- だから鵜飼は澪を見て、ほんの一瞬だけ**既視感**を覗かせる。「あなたも、いずれこうなる」＝**脅しでなく予言**として。

### 物語的効用

| 効用 | 内容 |
|---|---|
| **澪の選択に直接の重みを与える** | 澪が第3ブロックで**職務を踏み越えてナギを庇う**＝それは**「鵜飼にならない」という選択**になる。仲間化の決め手が、対敵構図と一本に繋がる。 |
| **非致死でも"怖い"を成立させる** | 殺さない追っ手が怖いのは、**「消されること」より「ああなること」**＝澪（と読者）が恐れるのは鵜飼の刃でなく**鵜飼の"無感"**。空洞化（削れ）の主題（[fab §4.5](fab_persona.md)）と響き合う。 |
| **議長センへの伏線**を人物で張る | 「葛藤を手続きで殺した個人」の極大版が議長セン（"安定のため"と本気で信じる）＝鵜飼は**末端でその縮図**を見せる。 |
| **ナギの主題とも交差** | ナギは「喪失だけは削れに渡さない」で人間性を保つ（[protagonist_core §0](protagonist_core.md)）。鵜飼は**渡しきってしまった者**＝ナギ・澪の双方への"こうはなるまい"の鏡。 |

### 開示の扱い（B1）

- 鵜飼の**過去（かつて葛藤した）は、B1 では"気配・既視感"まで**。フル開示でなく、**澪が「この人、どこか自分に似ている」と感じる違和感**として置く（[disclosure §2 仕込み](../series/disclosure_ledger.md) に新規埋設）。
- 完全隠蔽リスト（git機構・偽origin・4学派・喪失真相・道化正体・upstream）には**該当しない**＝人物の私的来歴ゆえ自由度はあるが、**naro の抑制（小出し）**で、本格的な背景回収は大弧I〜IIへ温存可。

## 2. 能力層（トレース＆抹消申請）

> [digital_tattoo §1](digital_tattoo.md)＝「形は力の象徴・強弱は能力層で決まる」／[battle §5 観測戦](battle_system.md)に準拠。鵜飼の力＝**戦う力でなく"見る力"と"消す権限"**。

| 要素 | 内容 |
|---|---|
| **型** | **観測者／射手型・外付け運用**。体制から託された**観測装置（鵜＝光のノード）**を放つ＝澪の「外付け＝支給品の弓」（[heroine §2](heroine_core.md)）の**監視特化版**。本人の身体能力・近接戦闘力は低い。 |
| **能力①トレース（足取り読み）** | 戻り標（帰還アンカー）の**残響**を辿る。放った鵜が動線に張りつき、**覚醒者の経路・立ち寄り先・拠点候補**を割り出す。**ナギの"寄り道する性質"（歩く exploit）が逆に足跡を増やす**＝[C-024](../series/canon_log.md) の弱点を突く。**※戻り標は時限で薄れる**（[dungeon §1.5-6](dungeon_design.md)）ため、鵜飼は**新鮮な足跡を捕まえねばならない＝時間圧力を負う追っ手**（永久追跡でない）。合流（戻り標の共有）が残す新鮮な足跡が彼の好餌＝[ep4](../plot/ep4_plot.md) で澪が合流した足跡を [ep5](../plot/ep5_plot.md) で読む。 |
| **能力②抹消申請** | 対象に「**なかったことにする**」烙印（強制 sync の申請）を打つ。**通れば拠点と深層活動の記録が revert**（[battle §4-2 crash＝非致死](battle_system.md)）。ただし**申請は独断で確定しない**＝**面側（登録・監視網）の裏取り・証言が要る**＝官僚機構ゆえの手続き的制約＝**最大の弱点であり、澪の効きどころ**。 |
| **拡張余地** | 「動線にピンを打って退路を塞ぐ」「複数の鵜で面を張り盲点を消す」等、**包囲・封鎖系**の派生（攻撃でなく"逃がさない"方向）。 |
| **弱点・コスト** | ①**動線が読めなければ無力**（デッドゾーン＝監視外の拠点[locations §4.5](locations.md) は中を覗けない）。②**盲目化・囮に弱い**（囮の戻り標でノードを誤誘導できる）。③**抹消申請に面側の裏取りが要る**（澪の報告保留・偽証で空振りする）。④本人が前に出れば**近接に脆い**（ナギの接近に弱い）。⑤**最大の天敵＝シード再訪**（[dungeon §1.5-6](dungeon_design.md)）：相手が**戻り標をやめてシードで再訪し始めると足跡が消え＝トレース不能**（"暗くなる(go dark)"）。戻り標の時限性ゆえ**時間が経てば古い足跡から消えていく**＝鵜飼は常に新鮮な足跡を追う消耗戦を強いられる。 |

## 3. 「出し抜く／なかったことにさせない」決着の機構

> [§1.5 暫定](../plot/book1_skeleton.md)を人物設計で確定。**撃破でなく出し抜く**が設計から必然になる。

1. **点 vs 面の連携**：ナギ＝点（デッドゾーンに隠れる歩く exploit）×澪＝面（監視網側）が完成形で噛む（[ep2 の軸](../plot/ep2_plot.md)）。
2. **シードで"暗くなる(go dark)"＝成長の到達点**（[dungeon §1.5-6](dungeon_design.md)）：ナギ（と澪）が**戻り標をやめてシード再訪に切り替える**＝**足跡そのものを断つ**。鵜飼のトレースの前提（新鮮な戻り標）が消える＝**追えなくなる**。これが第2ブロックの脅威（[ep5](../plot/ep5_plot.md)）に対する**機構的な答え**＝特訓・到達の payoff。
3. **囮の戻り標**：それでもナギは**偽の帰還アンカー**で鵜飼の鵜（観測ノード）を誘い出し、**本物の拠点をデッドゾーン化で盲目化**（古い足跡を逆用）。
4. **澪の踏み越え**：澪が**職務を踏み越え**、鵜飼の抹消申請の裏取り（面の証言）を**止める／囮へ誘導**＝**澪の正式仲間化の決め手**であり、同時に**「鵜飼にならない」という澪自身の選択**（[§1.5](#15-澪の未来の影鏡像としての芯)）。
5. **結果**：抹消申請が**空振り**。拠点と「澪が見逃し続けた記録」が**revert されずに残る**＝**なかったことにさせない＝巻1初の明確な勝利**。**鵜飼は退く（撃破でない）。去り際、澪に一瞥＝「あなたも、いずれ」**を残し、背後に "上"（4学派・統制機構）の影が差す＝巻末ブロックへ。

## 4. 既存設定・開示との整合

- **致死抑制（著者要件）**：抹消申請＝記憶・築いた現実の喪失止まり＝[C-003 非致死](../series/canon_log.md)と整合。殺さないが**"なかったことにする"**＝本作固有の恐怖。
- **語彙ルール**（[writing_style_notes](../guides/writing_style_notes.md)・[disclosure §0/§3](../series/disclosure_ledger.md)）：**表層プロースでは git/sync/revert 等の機構語を出さない**。本文で許される語＝「**記録を消される／なかったことにされる／戻される**」「**見られている／辿られている**」「**鵜（小さな光の鳥／無数の目）**」「**手続き／申請／烙印**」等の**現象・日常語**のみ。
- **拠点（デッドゾーン）との両立**：監視外の幻の駅（[C-029](../series/canon_log.md)）が**鵜飼に中を覗かれない**＝拠点獲得の価値が対敵構図で活きる。鵜飼は**入口の残響までは辿れるが、中は盲目**＝緊張の源泉。
- **FAB との対比**：FAB＝やさしいが温度のない誘惑（削れ）／鵜飼＝事務的で温度のない排除。**二種の"体温のなさ"**でナギを挟む（人＝澪の温度との三角）。
- **4学派の隠蔽**：鵜飼は無origin派・統制機構の末端だが、**本人も"上"を知らない**＝[disclosure](../series/disclosure_ledger.md)を破らず体制の存在だけを体感させる。
