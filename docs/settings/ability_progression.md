# 力の背骨：編集層の深化（しょぼい技 → おれつえー）規格書 v0.1

> 主人公の力の正体と成長の背骨。**「現実の記述を編集する」力**で、**編集できる層が深いほど強い**。
> 《晴れ間》＝最浅層の発露。究極＝基層（01/量子）の編集＝世界改変。**強さ＝編集層の深さ（質）＝数値インフレでない**。

## 1. 編集層ラダー（同じ力の"深さ"違い）

| 層 | 編集対象 | 例（技の見た目） | 性質・解禁目安 |
|---|---|---|---|
| **L0 表層の事実（自分含む）** | 起きた出来事・自他の状態の記録 | **軽い自己編集＝身体強化**（痛覚/疲労を"無かったこと"に・少し速く）／《軒下》濡れない／《晴れ間》足場を乾かす（状況技） | 一見しょぼい。**大弧I（1巻〜）**。容量小。**直接火力でなく"凌ぐ/いなす"** |
| **L1 物体の状態** | 運動・位置・温度・形 | 身体強化の強化（速さ・力）／落下を止める／物を一瞬留める／軽い攻撃の後押し | 大弧I 後半。応用・コンボで魅せる |
| **L2 場・力・時間** | 重力・電磁・**時間の流れ** | **局所重力を歪める**／時間を淀ませる（ダンジョンの時間歪みと同根） | **大弧II。心意級の縁**。代償が出始める |
| **L3 物理法則・定数** | 質量・重力定数・規則そのもの | **重力を極め質量を一点集中＝小さなブラックホール**／法則の上書き | **心意級 force-write**（致死・呪い）。大弧II〜III |
| **L4 基層＝01/量子** | 現実のソース（ビット/量子状態） | 現実を最小単位から書き換える＝世界改変の究極 | **おれつえー終局**。origin/upstream/マスターオリジン論争と直結。大弧III・続編 |

- **同一の力**：L0 も L4 も「記述の編集」。深さが違うだけ。だから《晴れ間》は弱いのでなく**浅い**。
- **重力もブラックホールも量子も別能力ではない**：L2→L3→L4 の連続。読者には「あの雨の子が、いつのまにか世界の底をいじっている」という驚き。

### 1.5 最初の実戦キット（べた・軽い身体強化＋軽い攻撃）

> 《晴れ間》（雨を消す/足場を乾かす）は**状況技**で、それ単体では戦えない。最初の戦闘力は地味で堅実なところから。

- **いちばん身近な"記録"＝自分自身**。だから最初の実用編集＝**自己への軽い編集＝身体強化**（"まだ動ける""痛くない""少し速い"を書き足す）。べたで地味だが堅実な土台。
- **攻撃**＝デジタルタトゥーで具現した**軽い武器**（剣・打撃＝力の象徴、digital_tattoo）＋編集の後押し。**最初は決定力に欠ける**。
- **環境編集（《軒下》《晴れ間》等）＝状況技**：守る・足場を作る・退路を開く。直接ダメージでなく**活路**を生む。
- → **1話の主人公は「倒す」より「凌ぐ・逃げる・いなす」**。**「弱いのに勝つ」＝身体強化＋状況編集＋相性・literacy・コンボの妙**（数値でない＝インフレ回避）。
- 成長＝この軽いキットが、編集層が深まる（L1物体→L2力…）につれ**重く・広く**なる。身体強化→物体操作→重力→…と、同じ"自己/物理の編集"が桁を上げていく。

## 2. なぜ徐々に開放されるのか（ゲート）

深い層は**理解（literacy）と適性（あざの段階）**が無いと触れない。獲得経路：

1. **literacy の深化**：世界の"書き方"を理解するほど深い層が見える（battle §2-3）。
2. **あざの段階開放**：あざ＝深層編集の異常アクセス。物語の節目で段階が開く（[anomaly_mark.md](anomaly_mark.md) の段階表と接続）。
3. **ダンジョン徒弟**：死者 master の技＝特定層の編集ノウハウを継ぐ（digital_tattoo §3/§8.5）。
4. **飛躍（ブレイクスルー）**：停滞→質的転換で一段深い層へ（battle §3.5-4）。

→ **数値で上がるのでなく、"触れる層が一段深くなる"**＝質的・段階的・不可逆寄り。お好みの「徐々に身につく/開放される」を満たす。

## 4. 表層での見え方と隠蔽（git 非表示・開示）

- **表層**：周囲には「天気を操る子」「物を止める子」に見える。**機構（記述の編集・層）は深層＝覚醒者の知。git 語は出さない。**
- **開示**：層の正体は段階的に明かす（[../series/disclosure_ledger.md](../series/disclosure_ledger.md)）。重力/ブラックホール/量子＝**長く伏せる仕込み**（早出し厳禁）。
- 読者向けの驚き設計：1巻「ただの天気いじり」→徐々に「これは現実そのものの編集では?」と気づかせる。

## 5. 第1話で蒔く仕込み（最小限）

- FAB「君は、さわっちゃいけない場所に手をのばしてる。だから書ける」（試作既出）＝**深層アクセスの示唆**。
