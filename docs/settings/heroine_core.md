## 1. なぜ「弓道狙撃＝監視役」なのか（設計の収束）

剣の少年案より、本作の機構と強く噛み合う：

- **第1話の最大の伏線と直結**：第1話は「**帰還の瞬間が一番脆い・戻り標で帰り道が割れている**」を首のうしろのぞわぞわで実装した（[locations.md §1](locations.md)・[battle_system.md §1-3](battle_system.md)）。**遠隔狙撃＝遠くから見て・帰り道を射抜く**は、この脆さを**具体的な脅威として顔を持たせる**。剣（近接）ではこの伏線と機構がズレていた。
- **タイトル「見られていた」が能力になる**：監視＝遠くから見る＝狙撃。第2話のテーマがそのままヒロインの力。
- **戦闘モデル（川原礫＝近接主体）の中で差別化**：遠隔狙撃キャラ＝**観測戦・間合いのドラマ**。近接バトル群のなかで絵的・戦術的に立つ。
- **関係性ドラマが監視に乗る**：「撃てる／撃たない」の一射ごとに、体制と心の葛藤が可視化される。敵でない障害＝重厚さ（naro 原則＝失敗・板挟みを描く）。
- **「洋の器に和の芯」＝主題対比**：山手の丘＝旧外国人居留地のカトリック・ミッション校（チャペル・規律＝洋の器）に通いながら、弓道（静・礼・間合い＝和の芯）を体現。ハーフ主人公（和×洋・どちらでも浮く・根を半分失った者）の**鏡像**＝**規律と信念に根ざした者**。容姿主題（外来者）と噛み合う。
- **山手＝舞台の符合**：主人公校（聖光学院モチーフ＝男子ミッション・山手駅最寄り）↔ ヒロイン校（横浜雙葉モチーフ＝女子ミッション・山手の丘）が**同じ"洋の器"の丘に並ぶ**（[locations.md §3](locations.md)）。欧米寄りハーフの主人公の舞台として最適。**校名はいずれもモチーフ化（作中は仮名）**。

## 1.5 「見逃し」を唐突にしない設計（三本の足場）

> 著者フィードバック（2026-06-14）＝「**2話の葛藤が唐突**」。監視役が突然見逃す＝ご都合主義（[editor-kawahara](../../.claude/skills/editor-kawahara/SKILL.md) が嫌う内面ロジック不足）を避けるため、見逃しを**心変わりでなく〈手順・規律・出来事〉から導く**。

| 足場 | 内容 |
|---|---|
| **① 手順** | 監視者の職務＝「見つけて・**確かめて**・報告する」。撃つ＝報告ではない。初遭遇の一射は**処刑でなく"読み"（観測テスト）**。即報告せず自分の目で確かめに来た時点で、撃たないのは弛みでなく**手順**。 |
| **② 弓道の規律（残心・正射）** | 「**見てから放つ**」。確かめ切らぬうちの報告＝**見る前に放つ**＝彼女自身の規律違反。報告の保留が性格と一貫。 |
| **③ イレギュラー共闘での目撃** | **第3勢力＝ダンジョンのイレギュラー（深層インスタンスの暴走）に二人とも巻き込まれる**（著者案）。澪はナギが**自分のためでなく澪を守って戦う**のを目撃＝"危険な潜り"の人物像と食い違う＝報告できない正当な理由。Ep2に戦闘の山場も生む。 |

- **弓と相性が悪いイレギュラー（著者案）**：暴走＝**点の無い混沌**（座標と瞬間が溶けて重なる／無数・面）。澪の弓＝**一点を射抜く（点の編集・戻り標を撃つ）**は撃つべき一点が無いと無力＝**相性最悪**。一方ナギ＝点でなく**"起きてしまった出来事（面・履歴）"に触れる**力＝混沌の継ぎ目を掴める。**点 vs 面**の能力対比で、登録狙撃の精密さが通じない領域を見せ、澪の「**登録＝安全／無登録＝危険**」の世界観に最初のひびを入れる＝見逃しの裏打ち（かつ、世界には体制が扱えない危険がある＝後巻への種）。
- **深い「なぜ」（鏡像・あざの気配・同種性）は B1 では気配のみ**（[disclosure_ledger §2](../series/disclosure_ledger.md)）。本文では「**指がすぐには引き金を引かない**」「なぜか整理がつかない」止まり＝後巻（関係性・あざ気づき）の回収用。

## 2. 能力層（魔法 vs 弓）

> [digital_tattoo.md §1](digital_tattoo.md)＝「形は力の象徴・強弱は能力層で決まる」に準拠。弓＝現実の弓矢でなく**遠隔ピンポイント編集の象徴**。

| 要素 | 内容 |
|---|---|
| **型** **[確定]** | **弓型タトゥー＝外付け型（支給品）**。体制（監視機関）から貸与された弓具に刻まれた外付けタトゥー＝**監視装置を持たされた末端**。返納・取り上げがありうる＝将来の火種。**ナギのヘッドホン（外付け・実は親の品）の鏡像**だが価値は正反対（澪＝体制の鎖／ナギ＝喪った親の愛）。 |
| **核** | **遠隔で一点を撃ち抜く＝対象の一点をピンポイント編集**。近接の面ではなく、遠隔の点。 |
| **対 戻り標** | **帰還アンカーを射抜ける**＝ダイバーの帰り道を破壊／封じる／マーキングする。第1話の脆さの体現。 |
| **対 技** | ナギがとっさに書いた小さな守り（《アンブレラ》）を、**遠くから撃って"無かったこと"にする／撃ち落とす**（剣案の巻き戻しを弓＝遠隔に移植）。 |
| **拡張余地** | 「撃って動きを止める（ピン留め）」「観測して座標を読む（探知）」等、監視・狙撃に連なる派生。 |
| **弱点・コスト** | 遠隔ゆえ**接近・遮蔽・速攻に弱い**（近接ライバル＝剣の少年を将来当てる余地）。一射ごとに集中・容量を要する＝乱射できない（naro＝コストを描く）。**最大の弱点＝「撃つべき一点が無い相手」**：座標が定まらない／無数／面の混沌には、点を射抜く弓は無力（第2話のイレギュラーで顕在化＝§1.5）。 |

## 3. 第1話・既存設定との整合

- **監視＝体制の末端**として提示（4学派・偽origin・"上"の正体は[disclosure 上限](../series/disclosure_ledger.md)で伏せる。B1 では「登録／無登録」「報告」「上」まで）。
- **拠点獲得との両立**：彼女に追われた結果、ナギは監視外の隠れ家（[locations.md §4.5](locations.md) 本牧の幻の地下鉄駅）へ。**狙撃から逃れる＝監視外へ**が動線として自然。
- **FAB 依存の入口との対比**：人の温度を持つ彼女（見逃す葛藤）↔ やさしいが温度のない FAB（削れの誘惑）。**人 vs AI** の対比軸を第2話で起動。
