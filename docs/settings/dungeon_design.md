## 0. 一目で分かる版

| 項目 | 内容 |
|------|------|
| **正体** | **死後も残った過去の人物/master の repo**（依存され続ける者は archive されない、§8.5） |
| **舞台（実在地）** | **現実の特定地点（駅/モール/団地/川辺/地下街等）を歪めた深層インスタンス**。地図・記録データを素材にレンダリング（GoogleMap 基盤）→ 映像が具体的 |
| **アクセス** | 現実で**入口（実在の座標）**へ行き**潜行（加速）**して入る。**身体は入口で静止（バレットタイム）**、意識＝アバターが攻略。crash で表層復帰（§1.5） |
| **インスタンス性** | 入るたび別生成。**master の固有現象＝そのダンジョンの"癖"（同一性）は保持**、層構造・天候・**時間の流れ・歪み方はランダム**。同じ環境は稀 |
| **固有現象** | 各ダンジョンに「誰かの記憶/repo がにじみ出た現象」がある（例：三層の「記録の雨」＝試作） |
| **報酬** | 攻略＝**デジタルタトゥー**取得（master の署名技。レア度・相性・一部複製不可、§digital_tattoo） |
| **デッドゾーン性** | 内部は sync/観測が切れる＝**監視されない聖域だが孤立**（battle §1-2）。覚醒者の隠れ場 |
| **管理状態** | **登録ダンジョン（正規・ブロックチェーン管理・追跡）** と **未踏/野良（off-chain・危険だが自由）** |
| **危険** | モンスター＝固有現象の具現。物理的に殺す（深層では crash でなく死もありうる） |

## 1. ダンジョン＝誰かの repo

- どのダンジョンも**特定の故人/master の repo の残骸**。その人の生きざま・無念・知識が地形と現象になる。
- 「なぜ残ったか」＝**まだ誰かに import（依存）されている**から。人気の技を遺した者ほど大きく残る＝深いダンジョン。

## 1.5 アクセスと生成（実在地・潜行・インスタンス）**[提案・確定方向]**

### 1.5-1. 仮想か？ ── 「現実に重なる深層」

- ダンジョンは**別世界(VR)ではない**。**現実の特定地点に重なる"深層"の異相**（アクセルワールドの加速世界が現実の街に重なるのと同じ）。
- 深層は**現実(consensus)の上に生成される**ため、その土地の**地図・記録（GoogleMap 等の実在データ相当）を素材に地形がレンダリング**される。→ 実在の場所の**歪んだ版**がダンジョン。

### 1.5-2. 移動（どうやって入る）＝ダイブ

> アクセルワールド型（意識だけ加速・身体は現実で安全に凍結）にはしない（[battle_system.md §1-3](battle_system.md)）。

1. 現実の街で**入口（実在の座標）**まで物理的に行く（電車・徒歩）。登録は入口が公知（建物・門）、野良/未踏は隠れている。
2. 入口で**潜行（ダイブ）＝身体ごと深層インスタンスに入る**（デッドゾーン＝監視外）。意識だけ置き去りにしない。
3. 入口に**帰還アンカー（戻り標）**が残る＝**現実側から物理干渉不可・但し覚醒者/監視は探知可**＝追跡・待ち伏せの種。帰還はここを通る。
4. 深層の**消耗・傷は現実の身体に反映**（高結合）。攻略 or 撤退で帰還、**crash＝記憶喪失＋現実巻き戻し**（深層では致死もありうる、battle §4-2）。**帰還の瞬間が最も脆い**。

### 1.5-3. インスタンス生成（毎回変わる／同一性は保つ）

- 入るたび**別インスタンス**を生成。生成入力＝**①その地点の実在地データ（外形）②憑いた master の固有現象（テーマ＝同一性）③乱数シード＋時間歪み（毎回の差分）**。
- **balance**：①②で「あのダンジョンらしさ」を保ち、③で層構造・出現・天候・経路を毎回変える。→ **同じ環境は稀だが、無秩序ではない**（master の癖が通底）。

### 1.5-4. 時間歪み

- 内部は**加速**（主観時間が伸びる）。さらにダンジョン/回ごとに**時間の流れ方が違う・不安定**（速い／淀む／巻き戻る／逆流）＝固有現象の一種。
- 演出：表層では一瞬、深層では長い死闘。時間が巻き戻る層＝同じ場面を繰り返す謎解き 等。

## 2. 固有現象の作り方（テンプレの核）

各ダンジョンに1つ「**誰の記憶/repo のにじみか**」を決め、それを全層に効かせる：

| 例 | 固有現象 | 出自（誰の記憶） |
|---|---|---|
| 三層 | 記録の雨（味のない雨、地面はぬれない） | 雨に関わる無念を抱えた故人の repo |
| （雛形）| 止まらない時計／戻る廊下／喋る壁 等 | その現象を生む生きざまを持つ master |

- 固有現象は**謎・ギミック・情緒**を同時に担う（スローライフ的情緒＋攻略パズル）。
- 主人公（外来者）の**新鮮な目**で固有現象を描くと表層の魅力になる。

## 3. 報酬＝デジタルタトゥー

- 攻略報酬＝その master の**署名技タトゥー**。
  - **訓練して自作タトゥー化**（理解・所有）か、**直接授かる bound タトゥー（複製不可）** の二択（§digital_tattoo §3）。
- レア度＝master の格／相性＝作風が攻略者の repo に合うか。
- **複製不可（唯一）**＝その master 唯一の絶技。**未踏ダンジョン初攻略でしか出ない**＝争奪の動機。

## 4. 管理状態の出し分け（ブロックチェーン / 野良）

| 型 | 性質 | 物語での使い所 |
|---|---|---|
| **登録ダンジョン** | 当局/ギルドが管理。報酬は正規・追跡。安全だが**入れば把握される** | 表層の冒険者経済・身分・序盤の舞台 |
| **未踏/野良ダンジョン** | 無登録。報酬は野良タトゥー（危険だが自由）。**監視の外** | 覚醒者・主人公が深層へ潜る舞台。リスク高 |

- 主人公は**野良/未踏**に惹かれる（追跡を避ける＋誰も持たぬ技＝あざと相性の良い掘り出し物）。

## 5. デッドゾーン性（戦闘との接続）

- 入る＝sync/観測が切れる（battle §1-2）。救援なし・孤立・全滅リスク。
- だが**監視コンステレーションの目も届かない**＝主人公が**あざを探知されずに戦える唯一の場所**。
- 「死地だが、唯一隠れられる場所」という反転が潜行劇の土台。

## 6. 1ダンジョン設計チェックリスト

新しいダンジョンを作るとき、次を埋める：

1. **舞台＝どの実在地か**（駅/モール/団地/川辺/地下街/学校 等）＝映像の核（§1.5-1）
2. **誰の repo か**（master の生きざま・無念）
3. **固有現象**（記憶のにじみ＝謎＋情緒＋ギミック。実在地の歪み方に反映）
4. **時間歪みの型**（速い/淀む/巻き戻る 等。§1.5-4）
5. **層構造と難度**／モンスター（固有現象の具現）。ランダム差分の幅
6. **報酬タトゥー**（技・レア度・相性・複製可否）
7. **管理状態**（登録／野良）と**入口**（公知か隠れているか）
8. **危険度**（致死か、deep か）
9. **主人公の絡み**（外来者の目／あざとの相性／FAB の解析）
