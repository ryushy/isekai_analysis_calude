# 横浜スタジアム風 monitor attack

澪が横浜スタジアム風の大型モニター上部から、バッターボックスへ攻撃を仕掛ける夜戦カットの検討案です。

共通ルール：

- 澪は現在レベル。射程は200〜300m圏内。
- 発射元は外野側の大型モニター/スコアボード上部。
- 目標はバッターボックス付近。
- 光る矢は映えるが、敵に位置がばれるため、目標まで伸びる連続レーザー状の射線は原則禁止。
- 体・顔・弓・目標方向の整合を重視する。
- 横浜スタジアムを直接再現しすぎず、読める文字・数字・ロゴ・広告は出さない。

## 次に直すなら

- 01/04：矢の残光をさらに短くし、連続した射線に見えないようにする。
- 03：澪の肩・顔・弓を明確にバッターボックスへ下向きに揃える。
- 全案：攻撃方向は「モニター上部 → バッターボックス」。矢が見える場合も、発射元までつながる光線にはしない。

## 反省点（2026-06-15 再整理）

- **小説背景の欠落**：通常の球場戦に寄り、現在がダンジョンであり現実と異なる状況だという前提が弱かった。
- **戦闘目的の欠落**：澪が単独でバッターを狙う絵になり、仲間を遠距離から支援する役割が見えなかった。近距離の仲間が敵の動きを止めた瞬間を撃つ、という連携を入れるべき。
- **隠密性の欠落**：大型モニター上で棒立ちだと目立ちすぎる。膝をつく、かがむ、構造物に身を隠す、影に沈むなどの工夫が必要。
- **矢の演出過多**：構えている矢と飛んでいる矢が同時に見える、複数の矢/光跡が出ると安っぽい。矢は「構え中の1本」「飛翔中の1本」「放った後の残心で矢なし」のどれかに絞る。
- **距離感の崩壊**：バッターボックス側から見た時、バックリーン/大型モニターとの距離や人物サイズのバランスが崩れていた。モニター上の澪は小さく、肩越しなら澪を大きく、というカメラ条件を明確化する。
- **球場地理の崩壊**：センター後方のモニター上部からホーム/バッターボックスへ狙う想定なのに、前後関係が逆転しているカットがあった。外野モニター→内野ホーム方向を固定する。
- **標的の誤り**：バッターを狙っているように見えるのは不適切。バッターボックスは位置指定であり、標的はその近くの敵/ダンジョン存在/仲間が止めた対象にする。
- **残心の不足**：放った後なら、飛翔線ではなく残心の形を見せるべき。澪の射手らしさは発射エフェクトより姿勢で出す。

## 再生成方針

- 舞台は「横浜スタジアム風ダンジョン夜戦」。
- 澪はセンター後方の大型モニター上部/保守通路にいる。
- 澪は棒立ちではなく、膝射ち・かがみ・身を隠す姿勢。
- 近距離の仲間がバッターボックス付近で敵を止めた瞬間、澪が一点を撃つ。
- 矢は1本だけ。カットごとに「つがえている」「飛んでいる」「残心」の状態を分ける。
- 敵にばれる連続した光線は描かない。
