# Yokohama Minatomirai station battle sheet

澪の軽量赤衣装 `mio_turnaround_light_crimson_functional_v1.png` を使った、横浜みなとみらい線風の地下鉄駅構内バトル用デザインシートです。

## 出力

- `outputs/mio_minatomirai_station_battle_sheet_base.png`: 文字なしのビジュアルベース。
- `outputs/mio_minatomirai_station_battle_sheet_v1.png`: 日本語の設定、ショットリスト、カメラ、アクション、台詞、照明、音響メモを載せた最終版。

## 意図

- 敵は黒い雨水、改札影、通勤者の輪郭、ガラス片が混ざった駅構内アノマリー。
- 澪は現在レベルの支援射手として、最初は「撃つべき一点」がない敵に不利。
- 退いて距離を作り、反射、改札光、水滴の揺れから弱点を一点化して撃つ。
- 連続レーザーや巨大ビームではなく、淡金の局所発光と一本の矢だけで見せる。
