# 横浜スタジアム風 monitor attack v2

前回案の反省を反映した再生成版です。

## 反映した修正

- 舞台を「現実の球場」ではなく、**横浜スタジアム風ダンジョン夜戦**として明記。
- 澪は単独でバッターを狙うのではなく、**近距離の仲間が敵の動きを止めた瞬間を遠距離支援で撃つ**。
- 澪は大型モニター上で棒立ちせず、**膝射ち・かがみ・保守通路/手すりを使った隠密姿勢**にする。
- 矢は1カットにつき1状態だけ：
  - 構え中の1本
  - 飛翔中の1本
  - 放った後の残心で矢なし
- 発射元までつながる連続レーザー状の射線は禁止。
- センター/外野側モニター上部から、内野ホーム/バッターボックス方向へ狙う地理を固定。
