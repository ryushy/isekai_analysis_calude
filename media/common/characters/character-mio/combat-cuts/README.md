# 澪 combat-cuts

篠宮澪の「現在レベル」の戦闘スタイル検討用カット集です。

採用キャラ参照：`../outputs/ref_sheet_v3_cute_uniform.png`

## 生成方針

- 現在の澪：射程はおよそ200〜300m。視認・座標・戻り標など、撃つべき一点が必要。
- 将来の澪：ランドマーク級の高所から観覧車内の敵を射抜くような、実質無限射程の一撃必殺アーチャー。
- 本カット集では、現在の弱さや戦術を残しつつ、将来アーチャーへの匂いを案ごとに変えています。

## 著者評価メモ（2026-06-15）

- 現時点で評価できるのは **02**。
- **05** は発想は良いが、目標に対する顔・体・弓の向きに注意。さらに目標まで光の線が伸びる表現は、敵に位置がばれる致命的欠陥になるため、今後は原則避ける。
- 将来アーチャーへの匂わせは、露骨な射線ではなく、構図・視線・風・都市スケール・遠景の観覧車で示す。

## 推奨

- **現在の有力案**：02
- **05の再生成条件**：体・顔・弓が目標方向に自然に揃うこと。敵にばれる連続した光線を描かないこと。
- **今後の共通禁止**：目標まで伸びる可視レーザー状の線、姿勢と目標方向が噛み合わない構図。
