## 現在の採用版

- `outputs/ref_sheet_v3_cute_uniform.png`

## 旧版

- `outputs/ref_sheet_v2_face_model_refined.png`: approved model photos の顔特徴を反映した版。制服の肩まわりが硬く見えたため v3 へ更新。
- `outputs/ref_sheet_v1.png`: 初期設定画。顔がやや汎用的だったため、approved model photos の顔特徴を反映した v2 へ更新。

## 戦闘カット候補

- `combat-cuts/`: 現在の澪の戦闘スタイル検討用カット集。将来のアーチャー像への匂わせ方を変えた複数案。

## 移動カット候補

- `mobility-cuts/urban-mobility-v1/`: 高速街移動の比較。自転車／スケートボード／インラインスケートを、表層制服と深層射手装束で検討。

## 衣装検討

- `costume-studies/three-style-collage-v1/`: 新コスチューム検討用の3スタイルコラージュ。
- `costume-studies/summer-seaside-three-style-v1/`: 真夏の海辺・海辺ダンジョン戦闘向けの3スタイルコラージュ。
- `costume-studies/adult-future-tattoo-three-style-v1/`: 成人後/未来版の澪を前提にした、高露出衣装とキー・タトゥーの3スタイルコラージュ。

## 標準装備

- `equipment/adult-future-standard-equipment-v1/`: 成人後/未来版の標準装備。通常弓、衣装連動弓、能力充填弓、変化後衣装、太ももタトゥー位置。
