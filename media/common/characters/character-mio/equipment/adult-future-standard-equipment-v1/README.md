# Mio equipment: adult future standard equipment v1

成人後/未来版の澪を前提にした標準装備資料です。
現行設定の16歳の澪ではなく、将来到達後の「コスチューム変化後」の装備として扱います。

出力:
- `outputs/mio_adult_future_standard_bow_v1.png`
- `outputs/mio_adult_future_standard_costume_props_v1.png`

## 固定設定

- タトゥー位置は **上腿外側から太もも正面寄り** に固定する。
- タトゥーは弓と星座を組み合わせた発光紋。物語上のキーで、衣装のスリットや戦闘中のずれで偶然見える。
- 普段の弓は普通の弓道の弓に見える。
- コスチューム変化時、弓も同時に黒赤金の戦闘形態へ変化する。
- 能力を込めるほど、弓の光沢が木・漆の質感から淡金の真珠光沢へ変わる。

## 画像別メモ

### Bow

左から順に、通常時、衣装連動時、能力充填時。
同じ弓の状態差として読みやすいので、標準装備の基準資料にできる。

### Costume Props

成人後/未来版の戦闘衣装を小道具として分解した資料。
太ももの発光タトゥーはこの位置で固定する。

採用時の注意:
- 実戦シーンではヒールを低くする、または足元をより機動寄りに調整してよい。
- 胸部の露出は成人後/未来版の衣装検討範囲。現在の澪へ逆輸入しない。
- 現在の澪に関連づける場合、タトゥーは肌露出ではなく護符、包帯下の痕跡、衣装裏の紋章として匂わせる。
