# Mio costume study: three-style collage v1

澪の新しい深層/戦闘コスチューム検討用コラージュです。

出力：

- `outputs/mio_three_style_costume_collage_v1.png`
- `outputs/mio_turnaround_semireal_v1.png`: 右案の黒×深紅ロングコートを主軸に、中央案の和装要素を少し統合した半実写寄りTurnaround。
- `outputs/mio_turnaround_white_kyudo_semireal_v1.png`: 中央案の白×深紅の正統弓道/巫女寄り衣装を主軸にした半実写寄りTurnaround。
- `outputs/mio_turnaround_light_crimson_functional_v1.png`: 白衣装の儀式感と重さを削り、添付の赤い深層衣装の片肩・巻き布・軽装レイヤーへ寄せた機能性重視Turnaround。

## 3案

| 位置 | 方向性 | 狙い | メモ |
|---|---|---|---|
| 左 | 隠密支援射手 | 現在のダンジョン戦闘に最も近い。膝射ち・低姿勢・支援狙撃向け | 実戦感が強い。光の弓演出は少し強めなので、採用時は矢/弦の光量を抑える余地あり |
| 中央 | 正統弓道/巫女寄り | 澪の「和の芯」「弓道の家系」を最も素直に出す | 品があり、ヒロイン感が強い。戦闘時の可動性をどう見せるかが課題 |
| 右 | 未来アーチャー予兆 | 屋上狙撃・将来の一撃必殺アーチャーの匂い | かっこよさが強い。現在形に使うなら強すぎるため、未来/中盤以降向け |

## 共通評価

- 顔・黒髪ポニーテール・華奢な体型はおおむね澪として保てている。
- 3案の差は明確で、衣装方向の比較資料として使いやすい。
- 文字ラベルは入れていないため、画像生成時の文字崩れは回避できている。
- 今後の単体再生成では、各案ごとに「矢は1本」「連続レーザー線は禁止」「体と顔の向き」を再指定する。
