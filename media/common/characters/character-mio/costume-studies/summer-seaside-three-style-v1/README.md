# Mio costume study: summer seaside three-style v1

真夏の海辺・海辺ダンジョン戦闘を想定した、澪の新コスチューム検討用コラージュです。

出力：

- `outputs/mio_summer_seaside_three_style_v1.png`

## 3案

| 位置 | 方向性 | 狙い | メモ |
|---|---|---|---|
| 左 | 潮影の実戦支援射手 | 岩場で膝射ち・支援射撃する現在戦闘寄り | 従来の緋色射手装束に近いが、海辺の実戦感はある。弓の光量はやや強め |
| 中央 | 白波の儀礼射手 | 夏の海辺と「和の芯」を両立する、白×緋の正統派 | ヒロイン感と清涼感が強い。実戦可動性は追加検討 |
| 右 | 夕潮の防水ロングコート射手 | 海辺の防水・防風・将来アーチャー予兆 | 未来寄りのかっこよさが強い。中盤以降の発展形として有力 |

## 共通評価

- 真夏の海辺らしい光、海、潮風の要素は出ている。
- 水着/ファンサービス方向ではなく、戦闘衣装として成立する範囲に留めた。
- 左は現在戦闘、中央は正統/儀礼、右は将来予兆として役割が分かれる。
- 今後単体化する場合は、矢1本ルール、姿勢と目標方向、連続レーザー禁止を再指定する。
