# character-nagi/

ナギの共通キャラ素材です。

## ワークフロー

1. 実在モデル写真を `references/model-photos/private/` に置く。
2. 生成に使ってよい写真を `references/model-photos/approved/` に選ぶ。
3. キャラ化プロンプトを `prompts/` に保存する。
4. 候補生成は `generations/run-YYYYMMDD_HHMMSS/` に残す。
5. 採用したキャラシートだけ `outputs/ref_sheet_v*.png` に置く。
