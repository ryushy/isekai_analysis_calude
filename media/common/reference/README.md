# common/reference/ — 共通参考画像（キャラ・横断利用）

> 複数話で使い回す参考画像置き場（プロジェクト共通）。
> **話固有の実在地参照**（コスモワールド・みなとみらい・山手学院など）は `media/book{N}/ep{M}/reference/` に置く。
> **案件固有の参照**は各 `…/storyboard/<case>/references/` に置く。主人公・キャラの顔参考は `media/common/characters/参考画像/` にも有り。

## 収録（共通キャラ・雰囲気参照）

| ファイル | 内容 | 用途 |
|---|---|---|
| `image.png` | 著者提示の参考画像（地下鉄駅＝秘密基地イメージ） | 秘密基地（[../backgrounds/honmoku-phantom-station-hideout/](../backgrounds/honmoku-phantom-station-hideout/)）の歪ませ元。キャラ雰囲気の参照にも使用 |
| `image_yoimiya.png` | 衣装・キャラの雰囲気参考 | コスチューム／キャラの雰囲気参照 |
| `スクリーンショット 2026-06-14 201207.png` | 篠宮澪の顔・体型リファレンス | 澪（モデル＝森脇リリカ／※実在人物・雰囲気参照に留め経歴等は断定しない） |
| `スクリーンショット 2026-06-14 201259.png` | 篠宮澪の顔・体型リファレンス（別カット） | 同上 |

## 命名規則

`<対象>_<出典メモ>.<ext>`（実在地は話固有フォルダ側で `<対象>_<出典メモ>.png`）。

> 画像はここ／`common/characters/` に置けば storyboard / seedance / character-sheet / stage-background スキルが見た目の寄せに使える（[../../../docs/settings/locations.md](../../../docs/settings/locations.md) の実在地モチーフと対応）。
